## Anatomy & Overview

### Pericardium
**Two-layered sac** surrounding the heart:
- **Visceral pericardium** (Epicardium) - Adherent to heart
- **Parietal pericardium** - Outer fibrous layer
- **Pericardial cavity:** Space between layers (15-50 ml fluid normally)

### Functions
- **Mechanical protection**
- **Prevents excessive cardiac dilation**
- **Reduces friction** (pericardial fluid)
- **Barrier to infection**

### Pericardial Diseases Classification

```
┌────────────────────────────────────────────────────┐
│      TYPES OF PERICARDIAL DISEASES                 │
├────────────────────────────────────────────────────┤
│ 1. ACUTE PERICARDITIS                              │
│    • Inflammation of pericardium                   │
│                                                    │
│ 2. PERICARDIAL EFFUSION                            │
│    • Excess fluid in pericardial cavity            │
│                                                    │
│ 3. CARDIAC TAMPONADE                               │
│    • Hemodynamic compromise from effusion          │
│                                                    │
│ 4. CONSTRICTIVE PERICARDITIS                       │
│    • Chronic fibrosis, impaired filling            │
│                                                    │
│ 5. EFFUSIVE-CONSTRICTIVE PERICARDITIS              │
│    • Effusion + Constriction                       │
└────────────────────────────────────────────────────┘
```

## Acute Pericarditis

### Definition
**Inflammation of the pericardium** characterized by **chest pain**, **pericardial friction rub**, and **ECG changes**.

### Etiology

**"CARDIAC"**
```
C - Collagen vascular disease (SLE, Rheumatoid arthritis, Scleroderma)
A - Autoimmune/Post-cardiac injury (Dressler's syndrome, Post-pericardiotomy)
R - Renal failure (Uremic pericarditis)
D - Drugs (Hydralazine, Procainamide, Isoniazid)
I - Idiopathic/Viral (Most common - 80-90%)
A - Aortic dissection (with hemopericardium)
C - Cancer (Lung, Breast, Lymphoma)

Additional:
- Infection (Bacterial - TB, Viral - Coxsackie, HIV)
- Myocardial infarction (Acute - 1-3 days, Dressler's - 2-10 weeks)
- Radiation
- Trauma
```

**Most Common Causes:**
- **Idiopathic/Viral:** 80-90% (Coxsackie B, Echovirus, Adenovirus)
- **Tuberculosis:** Most common in developing countries (India, Africa)
- **Post-MI:** Acute (1-3 days), Dressler's syndrome (2-10 weeks)
- **Uremia:** Chronic kidney disease

### Clinical Features

**Classic Triad:**
1. **Chest pain**
2. **Pericardial friction rub**
3. **ECG changes**

#### **1. Chest Pain - HALLMARK**

**Characteristics:**
```
┌────────────────────────────────────────────────────┐
│      PERICARDITIC CHEST PAIN                       │
├────────────────────────────────────────────────────┤
│ • SHARP, PLEURITIC (worse with breathing)          │
│ • RETROSTERNAL or Left precordial                  │
│ • Radiates to TRAPEZIUS ridge (Pathognomonic)      │
│ • Worse with:                                      │
│   - Lying supine                                   │
│   - Inspiration                                    │
│   - Coughing                                       │
│ • Better with:                                     │
│   - SITTING UP and LEANING FORWARD (Hallmark)      │
└────────────────────────────────────────────────────┘
```

**Differentiates from MI:**
- **Pleuritic** (worse with breathing)
- **Positional** (better sitting forward)
- **Radiates to trapezius** (Phrenic nerve irritation)

#### **2. Pericardial Friction Rub - PATHOGNOMONIC**

**Characteristics:**
- **High-pitched, scratchy, grating sound**
- **Best heard:** Left lower sternal border
- **Patient position:** Sitting, leaning forward, in expiration
- **Triphasic:** 3 components (Atrial systole, Ventricular systole, Ventricular diastole)
  - **Most common:** Biphasic or Monophasic
- **Transient:** May come and go
- **Present in 33-85%** of cases

**DIAGNOSTIC** if present (Specific for pericarditis)

#### **3. Other Symptoms**

- **Fever** (Low-grade)
- **Dyspnea** (if large effusion)
- **Cough**
- **Myalgia, malaise**

### Investigations

#### **ECG - KEY DIAGNOSTIC TOOL**

**4 Sequential Stages:**

```
┌────────────────────────────────────────────────────────┐
│      ECG STAGES IN ACUTE PERICARDITIS                  │
├────────────────────────────────────────────────────────┤
│                                                        │
│ STAGE 1 (Hours to days):                              │
│ • Diffuse ST elevation (concave upward)                │
│ • PR depression (most specific - atrial injury)        │
│ • Spares aVR, V1 (ST depression in aVR)                │
│                                                        │
│ STAGE 2 (Days):                                        │
│ • ST returns to baseline                               │
│ • T waves flatten                                      │
│                                                        │
│ STAGE 3 (Weeks):                                       │
│ • T wave inversion (diffuse)                           │
│                                                        │
│ STAGE 4 (Weeks to months):                             │
│ • ECG normalizes OR                                    │
│ • Persistent T wave inversion                          │
│                                                        │
└────────────────────────────────────────────────────────┘
```

**Key Features:**
- **Diffuse ST elevation** (All leads except aVR, V1)
- **Concave upward** (vs convex in MI)
- **PR depression** (Most specific for pericarditis - 80%)
- **No reciprocal ST depression** (unlike MI)
- **No Q waves** (unlike MI)

**Differentiating from MI:**
```
┌────────────────┬──────────────────┬───────────────────┐
│                │  PERICARDITIS    │  MI (STEMI)       │
├────────────────┼──────────────────┼───────────────────┤
│ ST elevation   │ Diffuse, Concave │ Regional, Convex  │
├────────────────┼──────────────────┼───────────────────┤
│ PR segment     │ DEPRESSION ✓     │ Normal            │
├────────────────┼──────────────────┼───────────────────┤
│ Reciprocal ST↓ │ NO               │ YES               │
├────────────────┼──────────────────┼───────────────────┤
│ Q waves        │ NO               │ YES (later)       │
├────────────────┼──────────────────┼───────────────────┤
│ T inversion    │ After ST normal  │ While ST elevated │
└────────────────┴──────────────────┴───────────────────┘
```

#### **Cardiac Biomarkers**

- **Troponin:** May be **mildly elevated** (15-30%) - Myopericarditis
- **CRP, ESR:** Elevated (inflammation)
- **WBC:** May be elevated

#### **Chest X-ray**

- **Usually normal** (acute pericarditis without effusion)
- **"Water bottle" heart:** If large pericardial effusion (>250 ml)
- **Cardiomegaly**

#### **Echocardiography**

- **Pericardial effusion** (if present)
- **Rule out tamponade**
- **LV function** (normal in pure pericarditis)
- **Exclude other cardiac pathology**

#### **Additional Tests (Based on Suspected Etiology)**

- **Tuberculosis:** Sputum AFB, Mantoux test, Pericardial fluid ADA (>40 U/L)
- **Uremia:** BUN, Creatinine
- **Autoimmune:** ANA, Anti-dsDNA (SLE), RF (Rheumatoid)
- **Malignancy:** Cytology of pericardial fluid

### Management

#### **Acute Pericarditis (Without Effusion)**

**First-Line: NSAIDs + Colchicine**

**1. NSAIDs:**
- **Ibuprofen** 600-800 mg TID (Preferred - fewer side effects)
- **Indomethacin** 25-50 mg TID (Alternative)
- **Aspirin** 750-1000 mg TID (Especially post-MI)
- **Duration:** Until symptom resolution + CRP normalization (Usually 1-2 weeks), then taper

**2. Colchicine:**
- **Dose:** 0.5-0.6 mg BD (if >70 kg) or OD (if <70 kg)
- **Duration:** 3 months
- **Benefits:**
  - ↓ Symptom duration
  - ↓ Recurrence risk (COPE, ICAP trials)
- **Side effect:** Diarrhea (common)

**3. Corticosteroids:**
- **Reserved for:**
  - **Refractory** to NSAIDs + Colchicine
  - **Contraindication** to NSAIDs (Active GI ulcer)
  - **Specific indications:** Autoimmune, Uremic pericarditis
- **Dose:** Prednisone 0.25-0.5 mg/kg/day
- **Avoid as first-line:** ↑ Recurrence risk

**4. Avoid:**
- **Anticoagulation** (if possible - risk of hemopericardium)
  - If essential (e.g., AF, prosthetic valve), monitor closely

#### **Activity Restriction**

- **Athletes:** Avoid competitive sports until:
  - Symptom resolution
  - CRP normalization
  - No effusion
  - Minimum 3 months

#### **Treat Underlying Cause**

- **Uremic:** Dialysis (Most effective)
- **TB:** Anti-tuberculous therapy (RIPE)
- **Bacterial:** Antibiotics
- **Autoimmune:** Corticosteroids, Immunosuppressants
- **Post-MI (Dressler's):** NSAIDs + Colchicine (Avoid corticosteroids if possible)

#### **Recurrent Pericarditis (15-30%)**

**Management:**
- **NSAIDs + Colchicine** (Higher dose, longer duration - 6 months)
- **Corticosteroids** (if refractory - slow taper over months)
- **Azathioprine, Methotrexate** (Steroid-sparing agents)
- **IVIG, Anakinra (IL-1 inhibitor)** (Refractory cases)
- **Pericardiectomy** (Last resort - chronic refractory)

## Pericardial Effusion

### Definition
**Excess fluid in the pericardial cavity** (>50 ml)

### Etiology
**Same as acute pericarditis** + Additional causes:
- **Hypothyroidism** (Myxedema)
- **Congestive heart failure**
- **Nephrotic syndrome**
- **Cirrhosis**

### Classification

**By Speed of Accumulation:**
- **Acute:** Rapid accumulation → Small volume causes tamponade (e.g., Trauma, Aortic dissection)
- **Chronic:** Slow accumulation → Large volume tolerated (e.g., Hypothyroidism, Malignancy)

**By Size:**
- **Small:** <10 mm (Echo-free space)
- **Moderate:** 10-20 mm
- **Large:** >20 mm
- **Very Large:** >20 mm + Compression of cardiac chambers

### Clinical Features

**Small Effusion:** Often asymptomatic

**Large Effusion:**
- **Dyspnea**
- **Chest discomfort**
- **Cough** (Compression of bronchus)
- **Dysphagia** (Compression of esophagus)
- **Hoarseness** (Compression of recurrent laryngeal nerve)
- **Hiccups** (Phrenic nerve irritation)

**Signs:**
- **Muffled heart sounds** (if large)
- **Ewart's sign:** Dullness at left lung base (Pericardial effusion compressing left lower lobe)

### Investigations

**ECG:**
- **Low voltage** (QRS <5 mm in limb leads)
- **Electrical alternans:** Beat-to-beat variation in QRS amplitude (Swinging heart)
  - **SPECIFIC for large effusion/tamponade**

**Chest X-ray:**
- **"Water bottle" heart:** Flask-shaped cardiac silhouette (if >250 ml)
- **Cardiomegaly**

**Echocardiography - DIAGNOSTIC:**
- **Echo-free space** around heart
- **Quantify:** Small, Moderate, Large
- **Distribution:** Circumferential vs Loculated
- **Signs of tamponade** (See below)

### Management

**Treat Underlying Cause:**
- **Hypothyroidism:** Thyroid hormone replacement
- **Uremia:** Dialysis
- **TB:** Anti-tuberculous therapy

**Pericardiocentesis:**
- **Indications:**
  - **Cardiac tamponade** (Therapeutic)
  - **Large effusion** with symptoms
  - **Diagnostic:** Unknown etiology (send fluid for analysis)

**Pericardial Fluid Analysis:**
- **Appearance:** Straw-colored (Transudate), Bloody (Malignancy, Trauma), Purulent (Bacterial)
- **Cell count, Differential**
- **Protein, LDH, Glucose**
- **Gram stain, Culture, AFB, TB culture**
- **Cytology** (Malignancy)
- **ADA** (Adenosine deaminase >40 U/L → TB)

**Pericardial Window/Pericardiectomy:**
- **Recurrent effusions**
- **Malignant effusions** (Sclerosing agents)

## Cardiac Tamponade

### Definition
**Life-threatening compression of the heart** by pericardial fluid → **Impaired cardiac filling** → **↓ Cardiac output**

**MEDICAL EMERGENCY**

### Etiology

**Acute (Rapid Accumulation):**
- **Trauma** (Penetrating, Blunt)
- **Aortic dissection** (Type A with hemopericardium)
- **Cardiac rupture** (Post-MI - free wall rupture)
- **Iatrogenic:** Post-cardiac catheterization, Pacemaker insertion

**Subacute/Chronic:**
- **Malignancy** (Lung, Breast, Lymphoma)
- **Tuberculosis**
- **Uremia**
- **Acute pericarditis** (any cause)

### Pathophysiology

```
Pericardial Fluid Accumulation
        ↓
↑ Intrapericardial Pressure
        ↓
Compression of Cardiac Chambers (RA, RV first)
        ↓
↓ Diastolic Filling
        ↓
↓ Stroke Volume, ↓ Cardiac Output
        ↓
Compensatory ↑ HR (Tachycardia)
        ↓
If Severe: CARDIOGENIC SHOCK
```

**Key Concept:** **Pericardium is non-compliant** - small acute accumulation (100-150 ml) can cause tamponade, but large chronic accumulation (1-2 L) may be tolerated.

### Clinical Features

**Beck's Triad (Classic - 33%):**
1. **Hypotension** (↓ Cardiac output)
2. **Elevated JVP** (↑ RA pressure)
3. **Muffled heart sounds** (Fluid around heart)

**Additional Signs:**

**Pulsus Paradoxus - HALLMARK:**
- **Exaggeration of normal** (>10 mmHg drop in SBP with inspiration)
- **Normal:** <10 mmHg drop
- **Tamponade:** >10 mmHg (often >20 mmHg)
- **Mechanism:** During inspiration → ↑ RV filling → Septal shift → ↓ LV filling → ↓ SBP

**How to Measure:**
1. Inflate BP cuff above systolic
2. Deflate slowly
3. Note SBP when **Korotkoff sounds heard only during expiration**
4. Continue deflating - Note SBP when sounds **heard throughout respiratory cycle**
5. **Difference** = Pulsus paradoxus

**Other Signs:**
- **Tachycardia** (Compensatory)
- **Tachypnea, Dyspnea**
- **Cool extremities** (Poor perfusion)
- **Kussmaul's sign:** Paradoxical ↑ JVP with inspiration (Less common in tamponade - more in constriction)

### Investigations

**ECG:**
- **Low voltage**
- **Electrical alternans** (Swinging heart) - **SPECIFIC**
- **Sinus tachycardia**

**Chest X-ray:**
- **Cardiomegaly** ("Water bottle" heart)

**Echocardiography - DIAGNOSTIC:**

**Diagnostic Signs:**
```
┌────────────────────────────────────────────────────┐
│      ECHO FINDINGS IN TAMPONADE                    │
├────────────────────────────────────────────────────┤
│ • Pericardial effusion (Circumferential)           │
│ • RA/RV diastolic collapse (MOST SENSITIVE)        │
│   - RA collapse >1/3 of cardiac cycle              │
│   - RV collapse in early diastole                  │
│ • IVC plethora (Dilated IVC >2 cm, <50% collapse)  │
│ • Respiratory variation in flow velocities:        │
│   - Mitral inflow ↓ >25% with inspiration          │
│   - Tricuspid inflow ↑ >40% with inspiration       │
│ • Swinging heart (in large effusion)               │
└────────────────────────────────────────────────────┘
```

**Cardiac Catheterization:**
- **Equalization of diastolic pressures:** RA ≈ RV ≈ PA diastolic ≈ PCWP
- Rarely done (Echo sufficient)

### Management

**EMERGENCY - REQUIRES IMMEDIATE DRAINAGE**

**Acute Stabilization:**
1. **IV Fluids:** Aggressive fluid resuscitation (↑ Preload → ↑ Filling)
2. **Avoid:**
   - **Diuretics** (↓ Preload → Worsen tamponade)
   - **Vasodilators** (↓ Afterload → Hypotension)
   - **Positive pressure ventilation** (↑ Intrathoracic pressure → ↓ Venous return)

**Definitive Treatment: Pericardiocentesis**

**Procedure:**
- **Approach:** Subxiphoid (Most common), Apical, Parasternal
- **Echo or Fluoroscopy-guided**
- **Drain fluid:** Immediate relief (Even small volume - 50-100 ml)
- **Leave catheter** in place (for ongoing drainage)

**Indications for Surgical Drainage:**
- **Hemopericardium** (Trauma, Aortic dissection, Post-MI rupture)
  - Pericardiocentesis may worsen bleeding
  - **Surgical exploration + Repair**
- **Purulent pericarditis** (Bacterial)
  - **Pericardial window or Pericardiectomy**
- **Recurrent effusions**

**Post-Procedure:**
- Monitor for:
  - **Re-accumulation**
  - **Complications:** Bleeding, Pneumothorax, Arrhythmias, Coronary/Chamber laceration

## Constrictive Pericarditis

### Definition
**Chronic inflammation** → **Pericardial fibrosis and calcification** → **Impaired diastolic filling** of ventricles

### Etiology

**Developing Countries:**
- **Tuberculosis** - **Most common** (50-70%)

**Developed Countries:**
- **Idiopathic/Viral** (40-50%)
- **Post-cardiac surgery** (Post-pericardiotomy, Post-radiation)
- **Post-pericarditis** (Recurrent)

**Other Causes:**
- **Radiation therapy** (Mediastinal - Hodgkin's lymphoma, Breast cancer)
- **Connective tissue disease** (SLE, Rheumatoid)
- **Malignancy**
- **Uremia**
- **Post-trauma**

### Pathophysiology

```
Chronic Pericardial Inflammation
        ↓
Fibrosis + Calcification (Thick, rigid pericardium)
        ↓
"Rigid Shell" around heart
        ↓
↓ Diastolic Compliance (Impaired filling)
        ↓
↑ Atrial Pressures (RA, LA)
        ↓
Systemic/Pulmonary Venous Congestion
        ↓
RIGHT HEART FAILURE (Predominant)
```

**Hemodynamics:**
- **Early rapid filling, then abrupt cessation** ("Dip and plateau" - Square root sign)
- **Equalization of diastolic pressures**
- **Ventricular interdependence** (Respiratory variation in filling)

### Clinical Features

**Insidious Onset** (Months to years)

**Symptoms:**
- **Dyspnea** (exertional)
- **Fatigue**
- **Right heart failure symptoms:**
  - **Peripheral edema** (Legs, ankles)
  - **Ascites** (Often **massive, out of proportion** to peripheral edema)
  - **Abdominal distension**
- **Orthopnea** (Less prominent than left heart failure)

**Signs:**

**Elevated JVP - HALLMARK:**
- **Prominent x and y descents** (Steep, rapid)
- **Kussmaul's sign:** ↑ JVP (or failure to fall) with inspiration - **CLASSIC**
  - Normally JVP ↓ with inspiration
  - Constriction: Rigid pericardium prevents normal filling → JVP ↑

**Other Signs:**
- **Hepatomegaly** (Pulsatile liver - if TR present)
- **Ascites** (May be massive)
- **Peripheral edema**
- **Pericardial knock:** Early diastolic sound (Abrupt cessation of filling)
  - Earlier than S3, higher-pitched

**"Pericardial knock" vs S3:**
- **Pericardial knock:** Earlier (0.09-0.12 s after A2), Higher-pitched
- **S3:** Later (0.14-0.16 s after A2), Lower-pitched

### Investigations

**ECG:**
- **Low voltage** (Pericardial thickening)
- **Non-specific ST-T changes**
- **AF** (Atrial enlargement)

**Chest X-ray:**
- **Pericardial calcification** (20-30%) - **BEST SEEN on lateral view**
  - **DIAGNOSTIC** if present (Specific for constrictive pericarditis)
- **Normal/Mildly enlarged heart** (Unlike dilated CM)
- **Pleural effusions**

**Echocardiography:**

```
┌────────────────────────────────────────────────────┐
│      ECHO IN CONSTRICTIVE PERICARDITIS             │
├────────────────────────────────────────────────────┤
│ • Thickened pericardium (>4 mm) - CT/MRI better    │
│ • Normal/Mildly enlarged atria                     │
│ • Normal ventricular size, EF                      │
│ • Septal bounce (Ventricular interdependence)      │
│ • Respiratory variation in flow velocities:        │
│   - Mitral inflow ↓ >25% with inspiration          │
│   - Tricuspid inflow ↑ >40% with inspiration       │
│ • IVC plethora (Dilated, <50% collapse)            │
└────────────────────────────────────────────────────┘
```

**Cardiac CT/MRI - BEST for Pericardium:**
- **Pericardial thickness >4 mm** (Normal: 1-2 mm)
  - **>6 mm:** Highly suggestive
- **Pericardial calcification** (CT best)
- **Conical/Tubular ventricles**

**Cardiac Catheterization:**

```
┌────────────────────────────────────────────────────┐
│      CATHETERIZATION IN CONSTRICTION              │
├────────────────────────────────────────────────────┤
│ • "Dip and plateau" (Square root sign)             │
│   - Early rapid ↓ in pressure, then plateau        │
│ • Equalization of diastolic pressures:             │
│   - RA ≈ RV diastolic ≈ PA diastolic ≈ PCWP       │
│   (Within 5 mmHg of each other)                    │
│ • RVEDP/RVSP >1/3 (>0.33)                          │
│ • Discordant RV/LV systolic pressures with respir  │
│   (Ventricular interdependence)                    │
└────────────────────────────────────────────────────┘
```

### Differentiation: Constriction vs Restriction

**Often challenging - Both have diastolic dysfunction**

```
┌──────────────┬────────────────┬───────────────────┐
│              │ CONSTRICTION   │ RESTRICTION (RCM) │
├──────────────┼────────────────┼───────────────────┤
│ Pericardium  │ Thick (>4 mm)  │ Normal            │
├──────────────┼────────────────┼───────────────────┤
│ Myocardium   │ Normal         │ Thick/Infiltrated │
├──────────────┼────────────────┼───────────────────┤
│ Calcification│ Common (20-30%)│ Rare              │
├──────────────┼────────────────┼───────────────────┤
│ Kussmaul sign│ PRESENT ✓      │ ABSENT            │
├──────────────┼────────────────┼───────────────────┤
│ Resp variation│ MARKED (>25%)  │ Minimal (<10%)    │
│ (Doppler)    │                │                   │
├──────────────┼────────────────┼───────────────────┤
│ BNP          │ Normal/Low     │ Elevated          │
├──────────────┼────────────────┼───────────────────┤
│ Biopsy       │ Not needed     │ Diagnostic        │
└──────────────┴────────────────┴───────────────────┘
```

**Key Differentiators:**
- **Kussmaul's sign:** Present in constriction, Absent in restriction
- **Respiratory variation:** More marked in constriction
- **Pericardial calcification:** Suggests constriction
- **BNP:** Low in constriction, High in restriction

### Management

**Medical (Limited Role):**
- **Diuretics:** Cautious (Reduce ascites/edema, but may ↓ preload too much)
- **Treat underlying cause:** TB (Anti-tuberculous therapy)

**Definitive Treatment: Pericardiectomy**

**Indications:**
- **Symptomatic** (NYHA class III-IV)
- **Hemodynamic compromise**

**Procedure:**
- **Surgical resection** of pericardium (Anterior and lateral walls, phrenic-to-phrenic)
- **High-risk surgery:** 5-10% mortality
- **Outcomes:**
  - Improvement in 60-90%
  - May take **months** for full benefit (Epicardial constriction may persist)

**Poor Prognostic Factors:**
- **Radiation-induced** (Myocardial fibrosis)
- **NYHA class IV**
- **Low cardiac output**
- **Severe calcification**
- **RV dysfunction**

## NEET PG High-Yield Points

### 🔥 MUST KNOW

1. **Acute Pericarditis - Classic Triad:**
   - **Chest pain** (Sharp, pleuritic, worse lying down, better sitting forward)
   - **Pericardial friction rub** (Triphasic, scratchy, PATHOGNOMONIC)
   - **ECG changes** (Diffuse ST elevation, PR depression)

2. **Pericarditis ECG:**
   - **Stage 1:** Diffuse ST elevation (concave), **PR depression** (Most specific)
   - **Diffuse** (All leads except aVR, V1)
   - **No reciprocal ST depression** (Unlike MI)
   - **No Q waves**

3. **Pericarditis Management:**
   - **NSAIDs + Colchicine** (First-line)
   - **Colchicine 3 months** (↓ Recurrence)
   - **Avoid corticosteroids** as first-line (↑ Recurrence)

4. **Cardiac Tamponade - Beck's Triad:**
   - **Hypotension**
   - **Elevated JVP**
   - **Muffled heart sounds**

5. **Tamponade - Pulsus Paradoxus:**
   - **>10 mmHg drop in SBP** with inspiration (HALLMARK)
   - **Mechanism:** Inspiration → ↑ RV filling → Septal shift → ↓ LV filling

6. **Tamponade Echo:**
   - **RA/RV diastolic collapse** (Most sensitive)
   - **IVC plethora** (Dilated, <50% collapse)
   - **Respiratory variation** in flow (Mitral ↓ >25%, Tricuspid ↑ >40%)
   - **Electrical alternans** (ECG) - Swinging heart

7. **Tamponade Management:**
   - **EMERGENCY: Pericardiocentesis**
   - **IV fluids** (↑ Preload)
   - **AVOID:** Diuretics, Vasodilators, Positive pressure ventilation

8. **Constrictive Pericarditis - Most Common Cause:**
   - **Developed:** Idiopathic/Viral, Post-cardiac surgery
   - **Developing:** **Tuberculosis** (India, Africa)

9. **Constrictive Pericarditis - Kussmaul's Sign:**
   - **↑ JVP (or failure to fall) with inspiration** - **CLASSIC**
   - **Rigid pericardium** prevents normal filling

10. **Constrictive Pericarditis - Pericardial Knock:**
    - **Early diastolic sound** (Abrupt cessation of filling)
    - Earlier and higher-pitched than S3

11. **Constrictive Pericarditis - Investigations:**
    - **CXR:** Pericardial calcification (20-30%) - **DIAGNOSTIC**
    - **CT/MRI:** Pericardial thickness >4 mm
    - **Catheterization:** Dip-and-plateau (Square root sign), Equalization of diastolic pressures

12. **Constrictive Pericarditis - Management:**
    - **Pericardiectomy** (Definitive treatment)

13. **Constriction vs Restriction:**
    - **Kussmaul sign:** Present in constriction, Absent in restriction
    - **Pericardial calcification:** Constriction
    - **BNP:** Low/Normal in constriction, High in restriction

### Clinical Scenarios

**Scenario 1:** Sharp chest pain, worse lying down, better sitting forward, radiates to trapezius, friction rub → **Acute pericarditis**

**Scenario 2:** Diffuse ST elevation, PR depression on ECG → **Acute pericarditis**

**Scenario 3:** Hypotension, elevated JVP, muffled heart sounds, SBP drop >10 mmHg with inspiration → **Cardiac tamponade** (Beck's triad + Pulsus paradoxus)

**Scenario 4:** RA collapse on echo, IVC plethora → **Cardiac tamponade** → **Emergency pericardiocentesis**

**Scenario 5:** Chronic dyspnea, ascites, JVP ↑ with inspiration (Kussmaul), pericardial calcification on CXR → **Constrictive pericarditis**

**Scenario 6:** Dip-and-plateau on catheterization → **Constrictive pericarditis** (or Restrictive cardiomyopathy)

**Scenario 7:** Post-MI patient (2 weeks) with pleuritic chest pain, fever → **Dressler's syndrome** (Post-MI pericarditis)

**Scenario 8:** CKD patient on dialysis with pericarditis → **Uremic pericarditis** → **Dialysis** most effective treatment

### Quick Revision Checklist

- [ ] Acute pericarditis: Chest pain (sharp, positional), Friction rub, ECG (ST elevation, PR depression)
- [ ] Pericarditis ECG: Diffuse ST↑, PR↓, No reciprocal ST↓, No Q waves
- [ ] Pericarditis treatment: NSAIDs + Colchicine (3 months)
- [ ] Tamponade: Beck's triad (Hypotension, ↑ JVP, Muffled sounds)
- [ ] Pulsus paradoxus: >10 mmHg drop with inspiration
- [ ] Tamponade echo: RA/RV collapse, IVC plethora
- [ ] Tamponade treatment: Emergency pericardiocentesis, IV fluids
- [ ] Constrictive: TB (developing), Kussmaul's sign, Pericardial knock
- [ ] Constrictive CXR: Pericardial calcification
- [ ] Constrictive catheterization: Dip-and-plateau, Equalization
- [ ] Constriction vs Restriction: Kussmaul (present vs absent), BNP (low vs high)
- [ ] Pericardiectomy for constrictive pericarditis

### Question 1
A 30-year-old male presents with sharp chest pain that worsens with lying down and improves when sitting forward. ECG shows diffuse ST elevation with PR depression. What is the most appropriate initial management?

**A)** Thrombolysis
**B)** NSAIDs + Colchicine
**C)** Corticosteroids
**D)** Pericardiocentesis

<details>
<summary>Click to reveal answer</summary>

**Answer: B) NSAIDs + Colchicine**

**Explanation:**
- **Diagnosis:** Acute pericarditis
  - **Chest pain:** Sharp, positional (worse lying, better sitting forward)
  - **ECG:** Diffuse ST elevation + PR depression

**Management:**
- **First-line:** **NSAIDs + Colchicine**
  - NSAIDs: Ibuprofen 600-800 mg TID or Aspirin 750-1000 mg TID
  - **Colchicine:** 0.5-0.6 mg BD for **3 months** (↓ Recurrence)

**Why not others:**
- **Thrombolysis:** For STEMI (Pericarditis mimics STEMI but is NOT MI)
- **Corticosteroids:** Avoid as first-line (↑ Recurrence risk) - Reserved for refractory or specific indications
- **Pericardiocentesis:** For tamponade or large symptomatic effusion (Not indicated in uncomplicated pericarditis)
</details>

### Question 2
A trauma patient has hypotension, elevated JVP, and muffled heart sounds. BP is 80/60 mmHg. What is the most appropriate immediate management?

**A)** IV fluids
**B)** Diuretics
**C)** Emergency pericardiocentesis
**D)** Pericardiectomy

<details>
<summary>Click to reveal answer</summary>

**Answer: C) Emergency pericardiocentesis**

**Explanation:**
- **Diagnosis:** **Cardiac tamponade** (post-trauma - likely hemopericardium)
- **Beck's Triad:**
  - Hypotension
  - Elevated JVP
  - Muffled heart sounds

**Management:**
- **EMERGENCY: Pericardiocentesis**
  - Definitive treatment
  - **Even small volume drainage** (50-100 ml) provides immediate relief

**Supportive:**
- **IV fluids** (↑ Preload) - Can give while preparing for pericardiocentesis

**AVOID:**
- **Diuretics** (↓ Preload → Worsen tamponade)

**Note:**
- **Hemopericardium (trauma)** may require **surgical exploration** (Pericardiocentesis can worsen bleeding)
- But in acute tamponade with shock, **pericardiocentesis is life-saving first step**
</details>

### Question 3
Which ECG finding is MOST SPECIFIC for acute pericarditis?

**A)** Diffuse ST elevation
**B)** PR depression
**C)** Electrical alternans
**D)** Low voltage

<details>
<summary>Click to reveal answer</summary>

**Answer: B) PR depression**

**Explanation:**
- **PR depression** is the **MOST SPECIFIC** ECG finding for acute pericarditis (80%)
- Mechanism: **Atrial injury** (Pericardium inflamed → Atrial epicardium involved)

**Other ECG findings in pericarditis:**
- **Diffuse ST elevation** (More sensitive but less specific)
- **No reciprocal ST depression** (Unlike MI)
- **Concave upward ST** (Unlike convex in MI)
- **No Q waves**

**Why not others:**
- **Diffuse ST elevation:** Sensitive but can occur in other conditions
- **Electrical alternans:** Specific for **large pericardial effusion/tamponade** (Swinging heart)
- **Low voltage:** Pericardial effusion, Obesity, COPD, Amyloidosis
</details>

### Question 4
A patient has dyspnea, ascites, and JVP that paradoxically rises with inspiration. Chest X-ray shows pericardial calcification. What is the most likely diagnosis?

**A)** Cardiac tamponade
**B)** Constrictive pericarditis
**C)** Restrictive cardiomyopathy
**D)** Dilated cardiomyopathy

<details>
<summary>Click to reveal answer</summary>

**Answer: B) Constrictive pericarditis**

**Explanation:**
- **Kussmaul's sign:** ↑ JVP with inspiration - **CLASSIC** for constrictive pericarditis
- **Pericardial calcification** on CXR - **DIAGNOSTIC** (20-30% of constriction cases)

**Other features:**
- Ascites (often out of proportion to peripheral edema)
- Right heart failure symptoms
- Pericardial knock (Early diastolic sound)

**Catheterization:**
- Dip-and-plateau (Square root sign)
- Equalization of diastolic pressures

**Treatment:**
- **Pericardiectomy** (Definitive)

**Why not others:**
- **Tamponade:** Would have hypotension, Kussmaul's sign LESS common
- **Restrictive CM:** NO Kussmaul's sign, No pericardial calcification
- **Dilated CM:** Systolic dysfunction, No Kussmaul, No calcification
</details>
