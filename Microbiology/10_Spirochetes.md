# Spirochetes

## Overview
Spirochetes are spiral-shaped bacteria with unique motility via axial filaments (endoflagella). Three major pathogenic genera cause important human diseases: **Treponema** (syphilis), **Borrelia** (Lyme disease, relapsing fever), and **Leptospira** (leptospirosis).

---

## GENERAL CHARACTERISTICS OF SPIROCHETES ⭐⭐⭐⭐⭐

```
Unique Features:
┌────────────────────────────────────────┐
│ MORPHOLOGY:                            │
│ • Spiral/helical shape                 │
│ • Thin, flexible                       │
│ • Too thin to see on Gram stain ⭐      │
│   (require special staining)           │
│                                        │
│ MOTILITY:                              │
│ • Axial filaments (endoflagella) ⭐     │
│   - Between inner/outer membranes      │
│   - Corkscrew rotation                 │
│   - Enables penetration of tissues     │
│                                        │
│ VISUALIZATION:                         │
│ • Dark-field microscopy ⭐              │
│ • Fluorescent antibody staining        │
│ • Silver staining (Warthin-Starry)     │
│ • Giemsa stain (Borrelia only)         │
└────────────────────────────────────────┘

Pathogenic Genera:
1. Treponema - Syphilis
2. Borrelia - Lyme disease, relapsing fever
3. Leptospira - Leptospirosis
```

## 1. TREPONEMA PALLIDUM ⭐⭐⭐⭐⭐

### Microbiology
```
┌────────────────────────────────────────┐
│ • Spirochete (tightly coiled)          │
│ • Thin (0.1-0.2 μm) - too thin for     │
│   light microscopy                     │
│ • Microaerophilic                      │
│ • Slow generation time (30 hours)      │
│ • CANNOT be cultured in vitro ⭐⭐⭐⭐   │
│   (diagnosis is serologic)             │
│ • Visualized by:                       │
│   - Dark-field microscopy ⭐            │
│   - Direct fluorescent antibody (DFA)  │
└────────────────────────────────────────┘
```

### Transmission
- **Sexual contact** (most common)
- **Vertical** (transplacental - congenital syphilis)
- **Blood transfusion** (rare)
- **Direct contact** with infectious lesions

### Clinical Manifestations: Acquired Syphilis

#### Primary Syphilis ⭐⭐⭐⭐⭐
```
TIMING: 3 weeks after exposure (10-90 days)

CHANCRE: ⭐⭐⭐⭐⭐
┌────────────────────────────────────────┐
│ CHARACTERISTICS:                       │
│ • Single (usually, can be multiple)    │
│ • PAINLESS ⭐⭐⭐⭐⭐                     │
│   (vs chancroid: painful)              │
│ • Indurated (hard) base                │
│ • Clean, non-purulent                  │
│ • Round, regular borders               │
│ • Raised margins                       │
│ • Size: 0.5-2 cm                       │
│                                        │
│ LOCATION:                              │
│ • Genital (most common):               │
│   - Penis (glans, shaft)               │
│   - Vulva, cervix, vagina              │
│ • Extragenital:                        │
│   - Oral (lips, tongue)                │
│   - Anal                               │
│   - Fingers                            │
│                                        │
│ LYMPHADENOPATHY: ⭐                     │
│ • Regional, bilateral                  │
│ • Firm, rubbery, non-tender            │
│ • Non-suppurative                      │
│                                        │
│ RESOLUTION:                            │
│ • Self-limited (3-6 weeks)             │
│ • Heals WITHOUT scarring ⭐             │
│ • With or without treatment            │
│   (but organism persists!)             │
└────────────────────────────────────────┘
```

#### Secondary Syphilis ⭐⭐⭐⭐⭐
```
TIMING: 2-8 weeks after chancre (if untreated)

SYSTEMIC DISEASE - "Great Imitator"
┌────────────────────────────────────────┐
│ CONSTITUTIONAL:                        │
│ • Fever, malaise                       │
│ • Headache                             │
│ • Generalized lymphadenopathy ⭐        │
│   (epitrochlear nodes - highly         │
│    suggestive)                         │
│ • Weight loss                          │
│                                        │
│ DERMATOLOGIC (90%): ⭐⭐⭐⭐⭐           │
│                                        │
│ MACULOPAPULAR RASH: ⭐⭐⭐⭐⭐           │
│ • Copper/rose-colored                  │
│ • Symmetric, widespread                │
│ • NON-PRURITIC ⭐                       │
│ • Trunk, extremities                   │
│ • PALMS and SOLES ⭐⭐⭐⭐⭐             │
│   (highly characteristic!)             │
│   "From head to toe, including         │
│    palms and soles"                    │
│ • Does NOT spare mucosal surfaces      │
│                                        │
│ CONDYLOMA LATA: ⭐⭐⭐⭐                 │
│ • Moist, flat-topped papules/plaques   │
│ • Gray-white                           │
│ • HIGHLY INFECTIOUS (teeming with      │
│   treponemes)                          │
│ • Locations: Perineum, axilla,         │
│   intertriginous areas                 │
│ • vs Condyloma acuminata (HPV):        │
│   - Cauliflower-like, dry              │
│                                        │
│ MUCOUS PATCHES: ⭐                      │
│ • Painless, gray-white patches         │
│ • Oral mucosa, tongue, genitals        │
│ • "Moth-eaten" appearance              │
│ • Highly infectious                    │
│                                        │
│ PATCHY ALOPECIA: ⭐                     │
│ • "Moth-eaten" scalp                   │
│ • Eyebrows, beard                      │
│                                        │
│ OTHER:                                 │
│ • Hepatitis (elevated transaminases)   │
│ • Nephrotic syndrome (membranous GN)   │
│ • Anterior uveitis, optic neuritis     │
│ • Meningitis (5-10%)                   │
│ • Osteitis, periostitis                │
│                                        │
│ RESOLUTION:                            │
│ • Self-limited (weeks-months)          │
│ • Enters latent phase                  │
└────────────────────────────────────────┘
```

#### Latent Syphilis ⭐⭐⭐⭐
```
ASYMPTOMATIC INFECTION
Diagnosed only by positive serology

EARLY LATENT:
• <1 year duration (or <2 years per CDC)
• Higher infectivity
• Can have secondary relapses (25%)

LATE LATENT:
• >1 year duration
• Low infectivity (except vertical
  transmission)
• ~1/3 progress to tertiary syphilis
  if untreated
```

#### Tertiary Syphilis ⭐⭐⭐⭐⭐
```
TIMING: Years to decades after infection
OCCURS: In ~1/3 of untreated patients

THREE MAIN FORMS:
┌────────────────────────────────────────┐
│ 1. GUMMATOUS SYPHILIS (15%):           │
│    • Granulomatous lesions (gummas)    │
│    • Skin, bones, liver, other organs  │
│    • Destructive but benign            │
│    • Caseous necrosis                  │
│    • Respond well to treatment         │
│                                        │
│ 2. CARDIOVASCULAR SYPHILIS (10%): ⭐⭐⭐│
│    Onset: 10-30 years                  │
│                                        │
│    ASCENDING AORTIC ANEURYSM: ⭐⭐⭐⭐⭐  │
│    • Thoracic aorta (ascending)        │
│    • "Tree-bark" appearance            │
│    • Aortic dilation                   │
│                                        │
│    AORTIC REGURGITATION (AR): ⭐⭐⭐⭐   │
│    • Dilation of aortic root           │
│    • Diastolic murmur                  │
│    • Heart failure                     │
│                                        │
│    CORONARY OSTIAL STENOSIS:           │
│    • Narrowing of coronary ostia       │
│    • Angina, MI                        │
│                                        │
│ 3. NEUROSYPHILIS (6-7%): ⭐⭐⭐⭐⭐      │
│    Can occur at ANY stage              │
│    (early or late)                     │
│                                        │
│    EARLY FORMS (<5 years):             │
│    • Asymptomatic neurosyphilis        │
│      - Abnormal CSF only               │
│    • Meningeal syphilis                │
│      - Headache, CN palsies            │
│    • Meningovascular syphilis          │
│      - Stroke (young patient)          │
│      - Endarteritis obliterans         │
│                                        │
│    LATE FORMS (>5 years):              │
│                                        │
│    GENERAL PARESIS: ⭐⭐⭐⭐             │
│    • Dementia (progressive)            │
│    • Personality changes               │
│    • Memory loss, judgment impairment  │
│    • Psychosis                         │
│    • "Paretic neurosyphilis"           │
│                                        │
│    TABES DORSALIS: ⭐⭐⭐⭐⭐            │
│    • Dorsal column degeneration        │
│    • LIGHTNING PAINS (sharp,           │
│      shooting leg pains) ⭐             │
│    • ATAXIA (sensory) ⭐                │
│      - Wide-based gait                 │
│      - Romberg positive                │
│    • Loss of proprioception,           │
│      vibration sense                   │
│    • ARGYLL ROBERTSON PUPILS ⭐⭐⭐⭐    │
│      - "Prostitute's pupil"            │
│      - Accommodates but doesn't react  │
│        to light                        │
│      - Small, irregular, bilateral     │
│    • Charcot joints (neuropathic       │
│      arthropathy)                      │
│    • Bladder dysfunction               │
│                                        │
│    OPTIC ATROPHY:                      │
│    • Vision loss, blindness            │
└────────────────────────────────────────┘
```

### Diagnosis ⭐⭐⭐⭐⭐

```
DIRECT DETECTION (Early Syphilis):
┌────────────────────────────────────────┐
│ DARK-FIELD MICROSCOPY: ⭐⭐⭐⭐          │
│ • From chancre or moist lesions        │
│ • See spirochetes (corkscrew motility) │
│ • Sensitivity: 80%                     │
│ • Not for oral lesions (commensal      │
│   treponemes present)                  │
│                                        │
│ DIRECT FLUORESCENT ANTIBODY (DFA):     │
│ • More sensitive than dark-field       │
│ • Can use on oral lesions              │
│                                        │
│ PCR:                                   │
│ • Increasingly available               │
│ • High sensitivity/specificity         │
└────────────────────────────────────────┘

SEROLOGIC TESTS (Most Common): ⭐⭐⭐⭐⭐
┌────────────────────────────────────────┐
│ TWO-TEST APPROACH:                     │
│                                        │
│ 1. NONTREPONEMAL TESTS (Screening): ⭐  │
│    Detect antibodies to cardiolipin    │
│                                        │
│    VDRL (Venereal Disease Research     │
│          Laboratory):                  │
│    • Flocculation test                 │
│    • Quantitative (titers)             │
│    • Used for CSF testing ⭐            │
│    • Monitor treatment response        │
│                                        │
│    RPR (Rapid Plasma Reagin):          │
│    • Similar to VDRL                   │
│    • Does not require microscope       │
│    • Quantitative                      │
│    • Screening test of choice ⭐        │
│                                        │
│    Characteristics:                    │
│    • Become positive 1-4 weeks after   │
│      chancre appears                   │
│    • Titers correlate with disease     │
│      activity                          │
│    • Decline/become negative after     │
│      treatment ⭐                       │
│    • FALSE POSITIVES (5-20%): ⭐⭐⭐⭐   │
│      - Pregnancy                       │
│      - Autoimmune diseases (SLE, APS)  │
│      - Infections (EBV, malaria, TB)   │
│      - IV drug use                     │
│      - Elderly                         │
│      - Recent vaccination              │
│                                        │
│ 2. TREPONEMAL TESTS (Confirmatory): ⭐⭐│
│    Detect antibodies to T. pallidum    │
│    antigens                            │
│                                        │
│    FTA-ABS (Fluorescent Treponemal     │
│            Antibody Absorption):       │
│    • Most sensitive                    │
│    • First to become positive          │
│    • Confirms diagnosis                │
│                                        │
│    TP-PA (T. pallidum Particle         │
│           Agglutination):              │
│    • Replaces FTA-ABS in many labs     │
│                                        │
│    EIA/CIA (Enzyme/Chemiluminescence   │
│             Immunoassays):             │
│    • Automated, high-throughput        │
│    • Increasingly used for screening   │
│                                        │
│    Characteristics:                    │
│    • Remain POSITIVE FOR LIFE ⭐⭐⭐⭐   │
│      (even after successful treatment) │
│    • Cannot differentiate active from  │
│      past treated infection            │
│    • Cannot monitor treatment response │
│    • Few false positives               │
│                                        │
│ INTERPRETATION: ⭐⭐⭐⭐⭐               │
│ ┌──────────┬────────────┬──────────┐  │
│ │ RPR/VDRL │ FTA-ABS/   │ Interpret│  │
│ │          │ Treponemal │          │  │
│ ├──────────┼────────────┼──────────┤  │
│ │ Negative │ Negative   │ No       │  │
│ │          │            │ syphilis │  │
│ ├──────────┼────────────┼──────────┤  │
│ │ Positive │ Positive   │ Syphilis │  │
│ │          │            │ (active/ │  │
│ │          │            │  treated)│  │
│ ├──────────┼────────────┼──────────┤  │
│ │ Positive │ Negative   │ False +  │  │
│ │          │            │ RPR/VDRL │  │
│ ├──────────┼────────────┼──────────┤  │
│ │ Negative │ Positive   │ Treated  │  │
│ │          │            │ syphilis,│  │
│ │          │            │ or very  │  │
│ │          │            │ early    │  │
│ └──────────┴────────────┴──────────┘  │
└────────────────────────────────────────┘

NEUROSYPHILIS DIAGNOSIS: ⭐⭐⭐⭐⭐
┌────────────────────────────────────────┐
│ Indications for LP:                    │
│ • Neurologic/ophthalmic symptoms       │
│ • Tertiary syphilis                    │
│ • Treatment failure                    │
│ • HIV coinfection (CD4 <350 or RPR ≥1:32)│
│ • Latent syphilis >1 year (controversial)│
│                                        │
│ CSF FINDINGS:                          │
│ • Pleocytosis (lymphocytic, >5 WBC/μL) │
│ • Elevated protein (>45 mg/dL)         │
│ • VDRL: Specific but insensitive ⭐     │
│   - Positive = diagnostic              │
│   - Negative doesn't rule out          │
│ • FTA-ABS: Sensitive but less specific │
│   - Negative = rules out               │
│                                        │
│ Diagnosis:                             │
│ • Reactive CSF VDRL = definitive ⭐     │
│ • Compatible clinical + CSF abnormalities│
│   + positive serum treponemal test     │
└────────────────────────────────────────┘
```

### Treatment ⭐⭐⭐⭐⭐

```
PENICILLIN G - Drug of Choice for All Stages
┌────────────────────────────────────────┐
│ PRIMARY, SECONDARY, EARLY LATENT:      │
│ • Benzathine Penicillin G 2.4 MU IM ⭐  │
│   SINGLE DOSE                          │
│                                        │
│ LATE LATENT, TERTIARY (non-neuro):     │
│ • Benzathine Penicillin G 2.4 MU IM    │
│   weekly × 3 doses ⭐                   │
│                                        │
│ NEUROSYPHILIS: ⭐⭐⭐⭐⭐                │
│ • Aqueous Penicillin G 18-24 MU/day IV │
│   (3-4 MU Q4h) × 10-14 days            │
│ • Alternative: Procaine penicillin     │
│   2.4 MU IM daily + probenecid 500 mg  │
│   PO QID × 10-14 days                  │
│                                        │
│ CONGENITAL SYPHILIS: ⭐⭐⭐⭐            │
│ • Aqueous Penicillin G 50,000 units/kg │
│   IV Q12h (week 1) then Q8h × 10 days  │
│ • Alternative: Procaine penicillin G   │
│   50,000 units/kg IM daily × 10 days   │
│                                        │
│ PREGNANCY: ⭐⭐⭐⭐⭐                     │
│ • PENICILLIN ONLY (same doses as above)│
│ • If penicillin allergic:              │
│   DESENSITIZATION required ⭐           │
│   (no safe alternatives)               │
│                                        │
│ NON-PREGNANT, PENICILLIN ALLERGIC:     │
│ • Doxycycline 100 mg PO BID × 14 days  │
│   (early) or × 28 days (late)          │
│ • Ceftriaxone 1-2 g IM/IV daily ×      │
│   10-14 days (alternative)             │
│ • NOT for neurosyphilis or pregnancy   │
└────────────────────────────────────────┘

JARISCH-HERXHEIMER REACTION: ⭐⭐⭐⭐
┌────────────────────────────────────────┐
│ • Acute febrile reaction within 24h    │
│   of treatment                         │
│ • Fever, chills, myalgias, headache    │
│ • Worsening of skin lesions            │
│ • Hypotension (can be severe)          │
│                                        │
│ Mechanism:                             │
│ • Release of treponemal antigens →     │
│   inflammatory cytokines (TNF-α, IL-6) │
│                                        │
│ More common in:                        │
│ • Early syphilis (primary, secondary)  │
│ • Higher organism burden               │
│                                        │
│ Management:                            │
│ • Self-limited (resolves 12-24h)       │
│ • Supportive (antipyretics, fluids)    │
│ • NOT an allergy - continue treatment! │
│ • Warn patients beforehand             │
│                                        │
│ Note: Also occurs with other           │
│ spirochetes (Lyme, leptospirosis)      │
└────────────────────────────────────────┘

FOLLOW-UP: ⭐⭐⭐⭐
• Nontreponemal titers (RPR/VDRL) at:
  - 6, 12, 24 months (early syphilis)
  - 6, 12, 18, 24 months (late/tertiary)
• Expect 4-fold decline in titer by 6-12 months
• "Serologic cure": ≥4-fold decline
• "Serofast": Titers remain low but stable
  (not treatment failure)
```

## 2. BORRELIA SPECIES ⭐⭐⭐⭐⭐

### 2A. Borrelia burgdorferi - Lyme Disease

#### Microbiology
```
┌────────────────────────────────────────┐
│ • Spirochete (loosely coiled)          │
│ • Larger than Treponema                │
│ • Microaerophilic                      │
│ • Cultured in special media (BSK)      │
│   - Slow growth (weeks)                │
│   - Not routine                        │
│ • Visualized: Giemsa, silver stain     │
└────────────────────────────────────────┘
```

#### Epidemiology & Transmission ⭐⭐⭐⭐⭐
```
VECTOR: Ixodes tick (deer tick, black-legged tick)
┌────────────────────────────────────────┐
│ • Ixodes scapularis (Northeast, Midwest USA)│
│ • Ixodes pacificus (West Coast USA)   │
│ • NYMPH stage most important ⭐         │
│   - Peak activity: May-July            │
│   - Size: Poppy seed (1-2 mm)          │
│   - Hard to detect                     │
│                                        │
│ TRANSMISSION:                          │
│ • Tick attachment >36-48 hours ⭐       │
│   required for transmission            │
│ • Spirochetes in tick midgut migrate to│
│   salivary glands during feeding       │
│ • Early removal prevents infection     │
│                                        │
│ RESERVOIRS:                            │
│ • White-footed mouse (Peromyscus)      │
│ • White-tailed deer (not infected,     │
│   but supports tick reproduction)      │
│                                        │
│ GEOGRAPHY (USA): ⭐                     │
│ • Northeast (Connecticut to Maryland)  │
│ • Upper Midwest (Wisconsin, Minnesota) │
│ • Pacific Northwest                    │
└────────────────────────────────────────┘
```

#### Clinical Manifestations: Three Stages

```
STAGE 1: EARLY LOCALIZED (3-30 days): ⭐⭐⭐⭐⭐
┌────────────────────────────────────────┐
│ ERYTHEMA MIGRANS (EM): ⭐⭐⭐⭐⭐         │
│ • Occurs in 70-80% of cases            │
│ • Pathognomonic for Lyme disease ⭐     │
│                                        │
│ CHARACTERISTICS:                       │
│ • Expanding red rash                   │
│ • "Bull's-eye" or "target" lesion ⭐    │
│   (central clearing)                   │
│   - But homogeneous rash also common   │
│ • Begins at tick bite site             │
│ • Expands over days (>5 cm diameter) ⭐ │
│ • Warm to touch                        │
│ • Usually NOT painful or pruritic      │
│ • May have central induration/vesicle  │
│                                        │
│ SYSTEMIC:                              │
│ • Fever, chills                        │
│ • Fatigue, malaise                     │
│ • Headache                             │
│ • Myalgias, arthralgias                │
│ • Lymphadenopathy                      │
│                                        │
│ RESOLUTION:                            │
│ • Spontaneous in 3-4 weeks             │
│   (even without treatment)             │
└────────────────────────────────────────┘

STAGE 2: EARLY DISSEMINATED (days-weeks): ⭐⭐⭐⭐
┌────────────────────────────────────────┐
│ DERMATOLOGIC:                          │
│ • Multiple erythema migrans lesions ⭐  │
│   (hematogenous spread)                │
│   - Smaller than primary               │
│   - No central tick bite               │
│                                        │
│ NEUROLOGIC (15%): ⭐⭐⭐⭐⭐              │
│ "Lyme Neuroborreliosis"                │
│                                        │
│ Classic Triad:                         │
│ 1. LYMPHOCYTIC MENINGITIS ⭐            │
│    • Headache, neck stiffness          │
│    • Mild, subacute                    │
│                                        │
│ 2. CRANIAL NEUROPATHY ⭐⭐⭐⭐           │
│    • CN VII palsy (most common) ⭐      │
│    • Bilateral in 25% (vs Bell's: rare)│
│    • Other CNs: III, IV, VI, V         │
│                                        │
│ 3. RADICULONEUROPATHY                  │
│    • Painful radiculitis               │
│    • Peripheral neuropathy             │
│                                        │
│ CARDIAC (5%): ⭐⭐⭐⭐                   │
│ • Lyme carditis                        │
│ • AV BLOCK (any degree) ⭐⭐⭐⭐         │
│   - 1st degree most common             │
│   - 3rd degree can occur (temporary    │
│     pacemaker may be needed)           │
│ • Myocarditis, pericarditis (rare)     │
│ • Typically resolves in weeks          │
│                                        │
│ MUSCULOSKELETAL:                       │
│ • Migratory arthralgias/myalgias       │
│ • Brief attacks (days)                 │
└────────────────────────────────────────┘

STAGE 3: LATE DISSEMINATED (months-years): ⭐⭐⭐⭐⭐
┌────────────────────────────────────────┐
│ ARTHRITIS: ⭐⭐⭐⭐⭐ (Most common late) │
│ • Occurs in 60% if untreated           │
│ • Onset: Months to years               │
│                                        │
│ CHARACTERISTICS:                       │
│ • Large joints (KNEE 90%) ⭐⭐⭐⭐       │
│ • Monoarticular or oligoarticular      │
│ • Intermittent attacks (weeks-months)  │
│ • Swelling (often massive effusion)    │
│ • Pain (moderate, less than swelling)  │
│ • Erosive changes rare                 │
│                                        │
│ Synovial fluid:                        │
│ • Inflammatory (>50,000 WBC/μL)        │
│ • PMN predominant                      │
│ • Negative culture                     │
│ • PCR for B. burgdorferi may be +      │
│                                        │
│ LATE NEUROLOGIC (rare):                │
│ • Chronic encephalomyelitis            │
│ • Encephalopathy                       │
│   - Cognitive dysfunction, memory loss │
│   - Mood changes, sleep disturbance    │
│ • Peripheral neuropathy                │
│ • "Chronic Lyme disease"               │
│   (controversial entity)               │
│                                        │
│ DERMATOLOGIC:                          │
│ • Acrodermatitis chronica atrophicans  │
│   (ACA) - rare in US, more in Europe   │
│   - Chronic skin atrophy (extremities) │
│   - Violaceous discoloration           │
└────────────────────────────────────────┘
```

#### Diagnosis ⭐⭐⭐⭐⭐
```
┌────────────────────────────────────────┐
│ CLINICAL DIAGNOSIS:                    │
│ • Erythema migrans in endemic area =   │
│   LYME DISEASE ⭐⭐⭐⭐⭐                │
│ • No testing needed if classic EM!     │
│ • Treat empirically                    │
│                                        │
│ SEROLOGIC TESTING (Two-Tier): ⭐⭐⭐⭐⭐ │
│                                        │
│ STEP 1: EIA/IFA (Screening)            │
│ • Detects IgM and/or IgG antibodies    │
│ • Sensitive but less specific          │
│                                        │
│ STEP 2: Western Blot (Confirmatory)    │
│ • If EIA/IFA positive or equivocal     │
│ • IgM: ≥2 of 3 bands positive          │
│ • IgG: ≥5 of 10 bands positive         │
│                                        │
│ TIMING: ⭐⭐⭐⭐                         │
│ • IgM: Appears 2-4 weeks after EM      │
│   - Peaks 3-6 weeks                    │
│ • IgG: Appears 4-6 weeks               │
│   - Remains elevated indefinitely      │
│                                        │
│ INTERPRETATION:                        │
│ • Early EM (<30 days): Often NEGATIVE ⭐│
│   - Don't delay treatment!             │
│ • Disseminated/late: Usually positive  │
│ • Positive serology does NOT prove     │
│   active infection (may be past)       │
│ • Cannot use to assess cure            │
│                                        │
│ FALSE POSITIVES:                       │
│ • Other spirochetes (syphilis)         │
│ • Autoimmune diseases                  │
│ • EBV, HIV                             │
│ • Western blot improves specificity    │
│                                        │
│ SPECIAL TESTS:                         │
│ • PCR: Synovial fluid (Lyme arthritis) │
│ • Culture: Rarely done (slow, difficult)│
│ • CSF: Intrathecal antibody production │
│   (neuroborreliosis)                   │
└────────────────────────────────────────┘
```

#### Treatment ⭐⭐⭐⭐⭐
```
┌────────────────────────────────────────┐
│ EARLY LOCALIZED/DISSEMINATED           │
│ (without CNS/cardiac): ⭐⭐⭐⭐           │
│                                        │
│ ADULTS:                                │
│ • Doxycycline 100 mg PO BID × 10-21d ⭐ │
│   (preferred, also covers Ehrlichia)   │
│ • Amoxicillin 500 mg PO TID × 14-21d   │
│ • Cefuroxime 500 mg PO BID × 14-21d    │
│                                        │
│ CHILDREN:                              │
│ • Amoxicillin 50 mg/kg/day ÷ TID       │
│ • Doxycycline 4 mg/kg/day ÷ BID        │
│   (if ≥8 years)                        │
│ • Duration: 14-21 days                 │
│                                        │
│ PREGNANT/LACTATING:                    │
│ • Amoxicillin (avoid doxycycline)      │
│                                        │
│ NEUROLOGIC DISEASE: ⭐⭐⭐⭐⭐           │
│ (meningitis, encephalitis, radiculopathy)│
│                                        │
│ • Ceftriaxone 2 g IV daily × 14-28d ⭐  │
│ • Alternatives:                        │
│   - Cefotaxime 2 g IV Q8h              │
│   - Penicillin G 18-24 MU/day IV       │
│                                        │
│ Facial palsy alone (no meningitis):    │
│ • Oral doxycycline (same as early)     │
│                                        │
│ CARDIAC DISEASE: ⭐⭐⭐⭐                │
│ • Outpatient (1st degree AV block):    │
│   - Doxycycline or amoxicillin PO ×    │
│     14-21 days                         │
│ • Hospitalize if:                      │
│   - PR >30 ms                          │
│   - 2nd/3rd degree AV block            │
│   - Myocarditis symptoms               │
│ • Ceftriaxone 2 g IV daily initially   │
│ • May need temporary pacemaker         │
│ • Transition to PO when improved       │
│                                        │
│ LYME ARTHRITIS: ⭐⭐⭐⭐                 │
│ • Doxycycline 100 mg BID × 28 days     │
│ • Amoxicillin 500 mg TID × 28 days     │
│ • If neurologic symptoms: IV ceftriaxone│
│                                        │
│ "Antibiotic-refractory" arthritis:     │
│ • Persistent despite 2 months oral or  │
│   1 month IV antibiotics               │
│ • Synovectomy, NSAIDs, DMARDs          │
│ • NO additional antibiotics            │
│                                        │
│ POST-TREATMENT LYME DISEASE SYNDROME:  │
│ • Persistent symptoms (fatigue, pain,  │
│   cognitive) after appropriate treatment│
│ • Serology remains positive            │
│ • NO benefit from additional antibiotics│
│ • Supportive care                      │
└────────────────────────────────────────┘
```

#### Prevention
```
TICK AVOIDANCE:
• Protective clothing (long pants, sleeves)
• Tuck pants into socks
• Light-colored clothing (see ticks)
• Permethrin on clothing
• DEET on exposed skin
• Tick checks after outdoor activities

TICK REMOVAL: ⭐⭐⭐⭐
• Fine-tipped tweezers
• Grasp close to skin
• Pull straight up (steady pressure)
• Do NOT twist or jerk
• Do NOT crush, burn, or apply substances
• Clean area with alcohol

POST-EXPOSURE PROPHYLAXIS: ⭐⭐⭐⭐
┌────────────────────────────────────────┐
│ Indications (ALL must be met):         │
│ • Tick attached ≥36 hours ⭐            │
│ • Tick removed <72 hours ago           │
│ • Endemic area (≥20% tick infection rate)│
│ • Ixodes scapularis tick (identified)  │
│ • No contraindication to doxycycline   │
│                                        │
│ Regimen:                               │
│ • Doxycycline 200 mg PO × 1 dose ⭐     │
│   (single dose)                        │
│                                        │
│ Efficacy: 87% prevention                │
│                                        │
│ Alternative: Watchful waiting          │
│ • Monitor for EM (28 days)             │
│ • Treat if develops                    │
└────────────────────────────────────────┘

VACCINE:
• Previous vaccine (LYMErix) discontinued
• New vaccines in development
```

│                                        │

│ • Severe headache                      │
│ • Myalgias, arthralgias                │
│ • Nausea, vomiting                     │

│ • Petechiae, jaundice                  │

## 3. LEPTOSPIRA INTERROGANS ⭐⭐⭐⭐⭐

### Microbiology
```
┌────────────────────────────────────────┐
│ • Spirochete (tightly coiled)          │
│ • Thin, flexible                       │
│ • Hooked ends ⭐                        │
│ • Obligate aerobe                      │
│ • Cultured in special media (slow)     │
│ • >250 serovars                        │
│ • Zoonotic ⭐                           │
└────────────────────────────────────────┘
```

### Epidemiology & Transmission ⭐⭐⭐⭐⭐
```
RESERVOIRS:
• Rodents (rats, mice) ⭐
• Dogs, cattle, pigs, horses
• Asymptomatic carrier animals
• Shed in urine

TRANSMISSION:
• Contact with contaminated WATER/SOIL ⭐⭐⭐⭐
  - Fresh water (rivers, lakes, puddles)
  - Soil saturated with animal urine
• Entry through:
  - Mucous membranes (eyes, nose, mouth)
  - Abraded/broken skin
• Occupational exposure ⭐:
  - Farmers, veterinarians
  - Sewer workers
  - Military personnel
• Recreational:
  - Swimming, wading in contaminated water
  - Water sports, adventure races

GEOGRAPHY:
• Worldwide (tropical/subtropical > temperate)
• Outbreaks after flooding, hurricanes
```

### Clinical Manifestations ⭐⭐⭐⭐⭐

```
BIPHASIC PATTERN:
┌────────────────────────────────────────┐
│ PHASE 1: LEPTOSPIREMIC (4-7 days): ⭐   │
│                                        │
│ • Abrupt high fever, chills            │
│ • Severe headache (frontal) ⭐          │
│ • Myalgias ⭐⭐⭐⭐⭐ (severe!)          │
│   - "Calf muscle pain" characteristic  │
│ • Conjunctival suffusion ⭐⭐⭐⭐        │
│   (redness without discharge)          │
│ • Photophobia                          │
│ • Nausea, vomiting                     │
│ • Hepatosplenomegaly                   │
│ • Rash (maculopapular, rare)           │
│                                        │
│ Blood/CSF cultures: Positive           │
│ Leptospires in blood, CSF              │
│                                        │
│ ↓                                      │
│ Brief remission (1-3 days)             │
│ ↓                                      │
│                                        │
│ PHASE 2: IMMUNE (weeks): ⭐             │
│                                        │
│ • Recurrence of fever (lower)          │
│ • Meningismus (aseptic meningitis)     │
│   - CSF: Lymphocytic pleocytosis       │
│   - Organisms cleared from CSF         │
│ • Uveitis                              │
│ • Rash                                 │
│                                        │
│ Antibodies appear                      │
│ Leptospires in urine (leptospiruria)   │
└────────────────────────────────────────┘

SEVERITY SPECTRUM:
┌────────────────────────────────────────┐
│ ANICTERIC LEPTOSPIROSIS (90%):         │
│ • Mild, self-limited                   │
│ • Flu-like illness                     │
│ • Full recovery                        │
│                                        │
│ WEIL'S DISEASE (5-10%): ⭐⭐⭐⭐⭐       │
│ Severe form with TRIAD:                │
│                                        │
│ 1. JAUNDICE ⭐⭐⭐⭐⭐                   │
│    • Hepatitis (elevated bilirubin)    │
│    • Mild transaminitis                │
│      (vs viral hepatitis: high ALT)    │
│                                        │
│ 2. ACUTE KIDNEY INJURY ⭐⭐⭐⭐          │
│    • Acute tubular necrosis            │
│    • Oliguria                          │
│    • Elevated Cr, BUN                  │
│    • May require dialysis              │
│                                        │
│ 3. BLEEDING ⭐⭐⭐⭐                     │
│    • Thrombocytopenia                  │
│    • Hemorrhage (pulmonary most serious)│
│    • Petechiae, ecchymoses             │
│    • GI bleeding, hemoptysis           │
│                                        │
│ PULMONARY HEMORRHAGE: ⭐⭐⭐⭐           │
│ • Severe, life-threatening             │
│ • Hemoptysis, respiratory failure      │
│ • ARDS                                 │
│ • High mortality                       │
│                                        │
│ OTHER:                                 │
│ • Myocarditis, arrhythmias             │
│ • Rhabdomyolysis                       │
│ • Aseptic meningitis                   │
│                                        │
│ MORTALITY: 5-40% (Weil's disease)      │
└────────────────────────────────────────┘
```

### Diagnosis ⭐⭐⭐⭐⭐
```
┌────────────────────────────────────────┐
│ CULTURE: ⭐                             │
│ • Gold standard but slow (weeks)       │
│ • Blood/CSF (first week)               │
│ • Urine (after first week)             │
│ • Special media (Fletcher, EMJH)       │
│                                        │
│ MICROSCOPY:                            │
│ • Dark-field: Blood, urine             │
│ • Low sensitivity                      │
│                                        │
│ SEROLOGY: ⭐⭐⭐⭐⭐                     │
│ Most commonly used                     │
│                                        │
│ MAT (Microscopic Agglutination Test):  │
│ • Gold standard serology               │
│ • Requires live organisms              │
│ • Paired sera (4-fold rise)            │
│ • ≥1:100 single titer suggestive       │
│   (endemic), ≥1:800 probable           │
│                                        │
│ ELISA (IgM):                           │
│ • Rapid, sensitive                     │
│ • Appears early (week 1)               │
│ • Most practical                       │
│                                        │
│ PCR:                                   │
│ • Blood (early), urine (later)         │
│ • Increasing availability              │
│                                        │
│ LABS (Weil's Disease):                 │
│ • Hyperbilirubinemia (conjugated) ⭐    │
│ • Mild transaminitis (ALT <200)        │
│ • Elevated Cr, BUN                     │
│ • Thrombocytopenia                     │
│ • Elevated CK (rhabdomyolysis)         │
│ • CSF: Lymphocytic pleocytosis         │
└────────────────────────────────────────┘
```

### Treatment ⭐⭐⭐⭐⭐
```
┌────────────────────────────────────────┐
│ MILD DISEASE:                          │
│ • Doxycycline 100 mg PO BID × 7 days ⭐ │
│ • Amoxicillin 500 mg PO TID × 7 days   │
│ • Azithromycin 500 mg daily × 3 days   │
│                                        │
│ SEVERE DISEASE (Weil's): ⭐⭐⭐⭐⭐      │
│ • Penicillin G 1.5 MU IV Q6h × 7 days  │
│ • Ceftriaxone 1 g IV daily × 7 days    │
│ • Doxycycline 100 mg IV BID × 7 days   │
│                                        │
│ SUPPORTIVE:                            │
│ • Fluids, electrolytes                 │
│ • Dialysis (if AKI)                    │
│ • Transfusions (if bleeding)           │
│ • Mechanical ventilation (if ARDS)     │
│                                        │
│ JARISCH-HERXHEIMER:                    │
│ • Can occur (less common than syphilis)│
│ • Supportive management                │
└────────────────────────────────────────┘
```

### Prevention
```
• Avoid contaminated water (especially
  with cuts/abrasions)
• Protective clothing, boots
• Rodent control
• Doxycycline prophylaxis (high-risk
  exposure, e.g., military exercises)
• Vaccine (animals only, not humans)
```

## HIGH-YIELD MNEMONICS

### 1. **Spirochete Visualization - "Dark Silver Giemsa"**
- **Dark**-field microscopy (all)
- **Silver** stain (Treponema, Leptospira)
- **Giemsa** (Borrelia)

### 2. **Primary Syphilis Chancre - "SIPHILIS"**
- **S**ingle
- **I**ndurated
- **P**ainless
- **H**eals without scar
- **I**ncubation 3 weeks
- **L**ymphadenopathy (bilateral, non-tender)
- **I**nfectious
- **S**erpiginous border

### 3. **Secondary Syphilis - "HAR on Palms/Soles"**
- **H**epatitis
- **A**lopecia (moth-eaten)
- **R**ash (palms and soles, non-pruritic)

### 4. **Tertiary Syphilis - "GAN"**
- **G**ummas
- **A**ortic aneurysm
- **N**eurosyphilis (tabes dorsalis, general paresis)

### 5. **Argyll Robertson Pupil - "Prostitute's Pupil"**
- **A**ccommodates
- **R**eacts to accommodation
- But **R**eflex to light is lost
- (Prostitute accommodates but doesn't react to light)

### 6. **Hutchinson's Triad (Congenital Syphilis)**
- **H**utchinson's teeth
- **I**nterstitial keratitis
- **8th** nerve deafness

### 7. **Lyme Disease Stages - "RASH → FACE → KNEE"**
- Stage 1: **R**ash (erythema migrans)
- Stage 2: **FACE** (facial palsy)
- Stage 3: **KNEE** (arthritis)

### 8. **Lyme Neuroborreliosis Triad - "MRC"**
- **M**eningitis
- **R**adiculoneuropathy
- **C**ranial neuropathy (CN VII)

### 9. **Leptospirosis - "WET CAM"**
- **W**ater exposure
- **E**yes (conjunctival suffusion)
- **T**hrombocytopenia
- **C**alf myalgias
- **A**KI
- **M**eningitis (aseptic)

### 10. **Weil's Disease Triad - "JKB"**
- **J**aundice
- **K**idney injury (AKI)
- **B**leeding

## NEET PG MCQS

### Q1. A patient has painless genital ulcer with indurated base and bilateral non-tender lymphadenopathy. Diagnosis?
A) Chancroid
B) Genital herpes
C) Primary syphilis
D) LGV

**Answer: C**
**Explanation**: Primary syphilis - painless chancre (vs chancroid: painful), indurated, bilateral non-tender lymphadenopathy.

---

### Q2. Rash on palms and soles in a sexually active patient. Most likely diagnosis?
A) Primary syphilis
B) Secondary syphilis
C) Psoriasis
D) Viral exanthem

**Answer: B**
**Explanation**: Secondary syphilis - characteristic maculopapular rash involving palms and soles, non-pruritic.

---

### Q3. "Prostitute's pupil" (accommodates but doesn't react to light) is seen in:
A) Diabetes
B) Neurosyphilis (tabes dorsalis)
C) Multiple sclerosis
D) Horner syndrome

**Answer: B**
**Explanation**: Argyll Robertson pupil in neurosyphilis (tabes dorsalis) - accommodates but no light reflex.

---

### Q4. Treatment of choice for neurosyphilis?
A) Benzathine penicillin G IM
B) Doxycycline PO
C) Aqueous penicillin G IV
D) Ceftriaxone PO

**Answer: C**
**Explanation**: Neurosyphilis requires IV penicillin G (18-24 MU/day) × 10-14 days (must achieve CSF levels).

---

### Q5. A 10-year-old with bull's-eye rash after camping trip. Which tick vector?
A) Dermacentor
B) Ixodes
C) Amblyomma
D) Rhipicephalus

**Answer: B**
**Explanation**: Lyme disease (erythema migrans) - transmitted by Ixodes tick (deer tick, black-legged tick).

---

### Q6. Most common joint affected in Lyme arthritis?
A) Ankle
B) Hip
C) Knee
D) Wrist

**Answer: C**
**Explanation**: Lyme arthritis - KNEE affected in 90% (large joint, monoarticular or oligoarticular).

---

### Q7. Single-dose prophylaxis for Lyme disease after tick bite?
A) Amoxicillin 500 mg
B) Doxycycline 200 mg
C) Ceftriaxone 1 g
D) Azithromycin 1 g

**Answer: B**
**Explanation**: Post-exposure prophylaxis: Doxycycline 200 mg single dose (if tick attached ≥36h, endemic area, removed <72h ago).

---

### Q8. Farmer with high fever, severe calf myalgias, and conjunctival suffusion after flooding. Diagnosis?
A) Dengue
B) Leptospirosis
C) Malaria
D) Typhoid

**Answer: B**
**Explanation**: Leptospirosis - water exposure, conjunctival suffusion, severe myalgias (especially calves).

---

### Q9. Weil's disease triad includes all EXCEPT:
A) Jaundice
B) Acute kidney injury
C) Thrombocytosis
D) Bleeding

**Answer: C**
**Explanation**: Weil's disease (severe leptospirosis) - jaundice, AKI, bleeding/thrombocytopenia (not thrombocytosis).

---

### Q10. Which spirochete CANNOT be cultured routinely?
A) Treponema pallidum
B) Borrelia burgdorferi
C) Leptospira interrogans
D) All can be cultured

**Answer: A**
**Explanation**: T. pallidum cannot be cultured in vitro - diagnosis is serologic. Borrelia and Leptospira can be cultured (though slow, difficult).

## REVISION CHECKLIST

### Must Know for NEET PG:
- [ ] Spirochetes: Spiral shape, thin (special staining needed), axial filaments, dark-field microscopy
- [ ] T. pallidum: Cannot culture, serologic diagnosis
- [ ] Primary syphilis: **Painless chancre**, indurated, heals without scar, bilateral non-tender lymphadenopathy
- [ ] Secondary syphilis: **Palms/soles rash** (non-pruritic), condyloma lata, mucous patches, generalized lymphadenopathy
- [ ] Tertiary syphilis: Gummas, **ascending aortic aneurysm**, aortic regurgitation, neurosyphilis
- [ ] Neurosyphilis: Tabes dorsalis (**lightning pains, ataxia, Argyll Robertson pupil**), general paresis (dementia)
- [ ] Argyll Robertson pupil: **Accommodates but doesn't react to light** ("prostitute's pupil")
- [ ] Congenital syphilis: Early (snuffles, rash, hepatosplenomegaly, osteochondritis), Late (**Hutchinson's triad**, saddle nose, saber shins)
- [ ] Hutchinson's triad: **Hutchinson's teeth + Interstitial keratitis + 8th nerve deafness**
- [ ] Syphilis serology: **RPR/VDRL** (screening, quantitative, monitor treatment) + **FTA-ABS/treponemal** (confirmatory, lifelong positive)
- [ ] Syphilis treatment: **Benzathine penicillin G 2.4 MU IM** (single dose for early, 3 weekly doses for late)
- [ ] Neurosyphilis treatment: **Aqueous penicillin G IV** × 10-14 days
- [ ] Pregnancy syphilis: **Penicillin only** (desensitize if allergic)
- [ ] Jarisch-Herxheimer reaction: Fever/chills within 24h of treatment (all spirochetes), not allergy, continue treatment
- [ ] Lyme disease: **Ixodes tick** (deer tick), transmission requires **≥36-48h attachment**
- [ ] Erythema migrans: **Bull's-eye rash, ≥5 cm**, expands, pathognomonic for Lyme
- [ ] Lyme neuroborreliosis: Meningitis + **CN VII palsy (bilateral 25%)** + radiculoneuropathy
- [ ] Lyme carditis: **AV block** (any degree)
- [ ] Lyme arthritis: **Knee** (90%), large joint, monoarticular/oligoarticular, months-years after infection
- [ ] Lyme serology: **Two-tier** (EIA/IFA → Western blot), often negative in early EM
- [ ] Lyme treatment: **Doxycycline 100 mg BID** (early), **ceftriaxone IV** (neuroborreliosis), 28 days for arthritis
- [ ] Lyme prophylaxis: **Doxycycline 200 mg single dose** (if tick ≥36h, endemic area, removed <72h)
- [ ] Leptospirosis: **Contaminated water** (rat urine), occupational/recreational
- [ ] Leptospirosis features: **Conjunctival suffusion**, severe **calf myalgias**, biphasic fever
- [ ] Weil's disease: **Jaundice + AKI + Bleeding**, pulmonary hemorrhage, high mortality
- [ ] Leptospirosis diagnosis: Serology (MAT, ELISA IgM), culture slow
- [ ] Leptospirosis treatment: **Doxycycline** (mild), **penicillin G or ceftriaxone IV** (severe)
- [ ] Relapsing fever: **Recurrent fevers** (3-10 episodes), **antigenic variation**, Borrelia
- [ ] All spirochetes visualized by **dark-field microscopy**

## CLINICAL PEARLS ⭐

1. **All spirochetes are too thin for routine light microscopy** - need dark-field or special stains
2. **T. pallidum cannot be cultured** - diagnosis is serologic
3. **Painless chancre = syphilis**, painful = chancroid/herpes
4. **Palms/soles rash = secondary syphilis** (highly characteristic)
5. **Condyloma lata (syphilis) vs condyloma acuminata (HPV)** - flat vs cauliflower
6. **Argyll Robertson pupil = neurosyphilis** (accommodates but no light reflex)
7. **RPR/VDRL titers correlate with disease activity** (monitor treatment)
8. **Treponemal tests (FTA-ABS) positive for life** (even after cure)
9. **Neurosyphilis requires IV penicillin** (achieve CSF levels)
10. **Pregnancy syphilis = penicillin only** (desensitize if allergic)
11. **Jarisch-Herxheimer is NOT allergy** - continue treatment
12. **Screen ALL pregnant women** for syphilis (first visit, 3rd trimester, delivery)
13. **Lyme tick needs ≥36-48h attachment** to transmit
14. **Erythema migrans alone = Lyme** (no serology needed, treat empirically)
15. **Lyme serology often negative early** - don't delay treatment for EM
16. **Bilateral facial palsy = think Lyme** (rare in Bell's palsy)
17. **Knee arthritis months after tick bite = Lyme**
18. **Single-dose doxycycline 200 mg** for Lyme prophylaxis (if criteria met)
19. **Conjunctival suffusion + calf myalgias + water exposure = leptospirosis**
20. **Weil's disease: Jaundice with mild transaminitis** (vs viral hepatitis: high ALT)
