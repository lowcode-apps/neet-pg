# Malaria

## Overview
Malaria is a life-threatening parasitic disease caused by *Plasmodium* species, transmitted by *Anopheles* mosquitoes. It remains a major global health problem, particularly in tropical and subtropical regions.

---

## PARASITOLOGY ⭐⭐⭐⭐⭐

### Causative Organisms
```
PLASMODIUM SPECIES (5 human species):
┌────────────────────────────────────────┐
│ 1. P. falciparum ⭐⭐⭐⭐⭐              │
│    • Most dangerous, highest mortality │
│    • Cerebral malaria, severe malaria  │
│    • Africa (90% of deaths)            │
│    • Can infect RBCs of all ages       │
│                                        │
│ 2. P. vivax ⭐⭐⭐⭐                     │
│    • Most common (outside Africa)      │
│    • Hypnozoites → relapses ⭐          │
│    • Prefers young RBCs (reticulocytes)│
│    • Usually not fatal                 │
│                                        │
│ 3. P. ovale ⭐⭐⭐                       │
│    • Africa, Pacific islands           │
│    • Hypnozoites → relapses            │
│    • Benign, rare                      │
│                                        │
│ 4. P. malariae ⭐⭐⭐                    │
│    • Worldwide, less common            │
│    • 72-hour (quartan) fever ⭐         │
│    • NO hypnozoites, NO relapses       │
│    • Can persist for decades           │
│    • Nephrotic syndrome (immune complex)│
│                                        │
│ 5. P. knowlesi ⭐⭐                      │
│    • Southeast Asia (Borneo, Malaysia) │
│    • Zoonotic (monkey parasite)        │
│    • 24-hour cycle (quotidian fever)   │
│    • Can be severe (high parasitemia)  │
└────────────────────────────────────────┘
```

### Vector
```
ANOPHELES MOSQUITO: ⭐⭐⭐⭐⭐
┌────────────────────────────────────────┐
│ • ONLY Anopheles transmits malaria ⭐   │
│   (NOT Aedes, NOT Culex)               │
│                                        │
│ CHARACTERISTICS:                       │
│ • FEMALE mosquito only (needs blood    │
│   for eggs) ⭐                          │
│ • Nocturnal feeding (dusk-dawn) ⭐      │
│ • Resting position: 45° angle to       │
│   surface ⭐                            │
│ • Spotted wings                        │
│                                        │
│ • Found in tropical/subtropical areas  │
│ • Breeds in clean, stagnant water      │
└────────────────────────────────────────┘
```

### Life Cycle ⭐⭐⭐⭐⭐

```
COMPLEX LIFE CYCLE (2 hosts):
┌────────────────────────────────────────┐
│ DEFINITIVE HOST: Anopheles mosquito ⭐  │
│ (Sexual reproduction - sporogony)      │
│                                        │
│ INTERMEDIATE HOST: Humans ⭐            │
│ (Asexual reproduction - schizogony)    │
└────────────────────────────────────────┘

IN MOSQUITO:
┌────────────────────────────────────────┐
│ 1. Mosquito ingests gametocytes        │
│    (from infected human blood)         │
│    ↓                                   │
│ 2. Fertilization in mosquito gut       │
│    Male + female gametocytes → zygote  │
│    ↓                                   │
│ 3. Zygote → ookinete → oocyst          │
│    (on gut wall)                       │
│    ↓                                   │
│ 4. Oocyst ruptures → SPOROZOITES ⭐     │
│    Migrate to salivary glands          │
│    ↓                                   │
│ 5. INFECTIVE STAGE: Sporozoites        │
│    Injected during next bite           │
│    (10-18 days in mosquito)            │
└────────────────────────────────────────┘

IN HUMAN:
┌────────────────────────────────────────┐
│ LIVER STAGE (Pre-erythrocytic): ⭐      │
│                                        │
│ 1. Sporozoites injected → bloodstream  │
│    ↓                                   │
│ 2. Enter HEPATOCYTES (liver cells) ⭐   │
│    ↓                                   │
│ 3. Multiply asexually → SCHIZONTS      │
│    ↓                                   │
│ 4. Rupture → MEROZOITES released       │
│    (6-15 days - incubation period)     │
│    ↓                                   │
│    • P. vivax/ovale: Some form         │
│      HYPNOZOITES ⭐⭐⭐⭐⭐              │
│      (dormant liver forms)             │
│      → Reactivate months-years later   │
│      → RELAPSES                        │
│    • P. falciparum/malariae: NO        │
│      hypnozoites, NO liver relapses    │
│    ↓                                   │
│ BLOOD STAGE (Erythrocytic): ⭐⭐⭐⭐⭐   │
│                                        │
│ 5. Merozoites invade RBCs              │
│    ↓                                   │
│ 6. Asexual multiplication in RBCs:     │
│    Ring → Trophozoite → Schizont       │
│    ↓                                   │
│ 7. RBC ruptures → release merozoites + │
│    toxins                              │
│    • Causes FEVER (periodicity) ⭐      │
│    • Hemolysis                         │
│    ↓                                   │
│ 8. Merozoites invade NEW RBCs          │
│    Cycle repeats (48 or 72 hours)      │
│    ↓                                   │
│ 9. Some merozoites → GAMETOCYTES ⭐     │
│    (sexual forms, male/female)         │
│    • Infective to mosquito             │
│    • NOT pathogenic to human           │
│    • Target of transmission-blocking   │
└────────────────────────────────────────┘

FEVER PERIODICITY: ⭐⭐⭐⭐⭐
┌────────────────────────────────────────┐
│ • TERTIAN (48-hour cycle): ⭐           │
│   P. falciparum, P. vivax, P. ovale    │
│   Fever every 3rd day (day 1, 3, 5...) │
│                                        │
│ • QUARTAN (72-hour cycle): ⭐           │
│   P. malariae                          │
│   Fever every 4th day (day 1, 4, 7...) │
│                                        │
│ • QUOTIDIAN (24-hour): ⭐               │
│   P. knowlesi                          │
│   Fever daily                          │
│                                        │
│ • P. falciparum: Often IRREGULAR       │
│   (multiple broods, continuous fever)  │
└────────────────────────────────────────┘
```

## CLINICAL MANIFESTATIONS ⭐⭐⭐⭐⭐

### Uncomplicated Malaria
```
CLASSIC MALARIAL PAROXYSM: ⭐⭐⭐⭐⭐
┌────────────────────────────────────────┐
│ THREE STAGES (lasts 6-10 hours):       │
│                                        │
│ 1. COLD STAGE (15 min - 1 hour):       │
│    • Intense CHILLS, rigor ⭐           │
│    • Feeling of extreme cold           │
│    • Patient shivers, teeth chatter    │
│    • Rising temperature                │
│    ↓                                   │
│ 2. HOT STAGE (2-6 hours):              │
│    • HIGH FEVER (40-41°C) ⭐⭐⭐⭐       │
│    • Flushed, dry skin                 │
│    • Severe headache ⭐                 │
│    • Nausea, vomiting                  │
│    • Tachycardia                       │
│    ↓                                   │
│ 3. SWEATING STAGE (2-4 hours):         │
│    • Profuse SWEATING ⭐                │
│    • Defervescence (temperature drops) │
│    • Exhaustion, weakness              │
│    • Patient falls asleep              │
│    ↓                                   │
│ Between paroxysms: Patient feels well  │
│ (relatively)                           │
└────────────────────────────────────────┘

OTHER SYMPTOMS:
• Myalgias, arthralgias
• Hepatosplenomegaly ⭐
  - Spleen palpable (chronic)
• Anemia (hemolytic) ⭐
• Jaundice (mild)
• Thrombocytopenia

INCUBATION PERIOD:
• P. falciparum: 7-14 days
• P. vivax: 12-18 days (can be months)
• P. ovale: 15-18 days
• P. malariae: 18-40 days (longest)
• P. knowlesi: 9-12 days
```

### Severe Malaria (P. falciparum) ⭐⭐⭐⭐⭐

```
WHO CRITERIA (ANY of the following):
┌────────────────────────────────────────┐
│ CLINICAL MANIFESTATIONS:               │
│                                        │
│ 1. CEREBRAL MALARIA: ⭐⭐⭐⭐⭐          │
│    • Impaired consciousness (GCS <11)  │
│    • Coma (unarousable)                │
│    • Seizures (generalized)            │
│    • Pathology: Sequestration of       │
│      parasitized RBCs in brain         │
│      capillaries                       │
│    • Mortality: 15-20%                 │
│    • Neurologic sequelae: 10% (children)│
│                                        │
│ 2. SEVERE ANEMIA: ⭐⭐⭐⭐               │
│    • Hb <5 g/dL or Hct <15%            │
│    • Hemolysis + bone marrow suppression│
│                                        │
│ 3. ACUTE RESPIRATORY DISTRESS: ⭐⭐⭐    │
│    • Pulmonary edema, ARDS             │
│    • Increased capillary permeability  │
│    • High mortality                    │
│                                        │
│ 4. ACUTE KIDNEY INJURY: ⭐⭐⭐⭐         │
│    • Cr >3 mg/dL                       │
│    • Acute tubular necrosis            │
│    • "Blackwater fever" (hemoglobinuria)│
│                                        │
│ 5. CIRCULATORY COLLAPSE/SHOCK: ⭐⭐⭐    │
│    • Hypotension (SBP <80 mmHg)        │
│    • "Algid malaria"                   │
│                                        │
│ 6. METABOLIC ACIDOSIS: ⭐⭐⭐⭐          │
│    • pH <7.35, bicarbonate <15 mEq/L   │
│    • Lactic acidosis                   │
│    • Poor prognostic sign              │
│                                        │
│ 7. HYPOGLYCEMIA: ⭐⭐⭐⭐                │
│    • Glucose <40 mg/dL                 │
│    • Consumption + quinine-induced     │
│      insulin release                   │
│    • Can mimic cerebral malaria        │
│                                        │
│ 8. JAUNDICE: ⭐⭐⭐                      │
│    • Bilirubin >3 mg/dL                │
│    • Hemolysis + hepatic dysfunction   │
│                                        │
│ 9. BLEEDING/DIC:                       │
│    • Spontaneous bleeding              │
│    • Thrombocytopenia                  │
│                                        │
│ 10. REPEATED GENERALIZED SEIZURES:     │
│                                        │
│ 11. PROSTRATION:                       │
│     • Extreme weakness, unable to sit/ │
│       walk                             │
│                                        │
│ LABORATORY:                            │
│ • HYPERPARASITEMIA: ⭐⭐⭐⭐⭐           │
│   >5% (or >250,000/μL) parasitized RBCs│
│   - Risk of severe disease             │
│   - Unique to P. falciparum            │
│     (can infect all RBCs)              │
│                                        │
│ • Hyperlactatemia (lactate >5 mmol/L)  │
└────────────────────────────────────────┘

BLACKWATER FEVER: ⭐⭐⭐⭐
• Massive intravascular hemolysis
• Dark urine (hemoglobinuria) - "black water"
• Acute kidney injury
• Associated with:
  - Severe P. falciparum
  - G6PD deficiency + oxidant drugs
  - Quinine treatment (historically)
```

## DIAGNOSIS ⭐⭐⭐⭐⭐

```
GOLD STANDARD: MICROSCOPY ⭐⭐⭐⭐⭐
┌────────────────────────────────────────┐
│ BLOOD SMEAR EXAMINATION:               │
│                                        │
│ THICK SMEAR: ⭐⭐⭐⭐                    │
│ • Screening for parasites              │
│ • 10-40× more sensitive than thin      │
│ • Detects LOW parasitemia              │
│ • Cannot speciate reliably             │
│ • RBCs lysed (concentrated)            │
│                                        │
│ THIN SMEAR: ⭐⭐⭐⭐                     │
│ • SPECIES IDENTIFICATION ⭐             │
│ • Quantify parasitemia (%)             │
│ • See parasite morphology clearly      │
│ • RBCs intact                          │
│                                        │
│ STAINING: Giemsa, Field's             │
│                                        │
│ TIMING:                                │
│ • Obtain during fever spike (highest   │
│   parasitemia)                         │
│ • Repeat every 12-24h if negative but  │
│   high suspicion (3× total)            │
└────────────────────────────────────────┘

MORPHOLOGIC FEATURES: ⭐⭐⭐⭐⭐
┌────────────────────────────────────────┐
│ P. FALCIPARUM:                         │
│ • Ring forms ONLY (in peripheral blood)⭐│
│   - Other stages sequestered           │
│ • Multiple rings per RBC ⭐             │
│ • "Headphone" or "appliqué" forms      │
│ • Banana-shaped gametocytes ⭐⭐⭐⭐     │
│   (crescent, pathognomonic)            │
│ • Maurer's dots/clefts (RBC cytoplasm) │
│                                        │
│ P. VIVAX:                              │
│ • All stages in peripheral blood       │
│ • ENLARGED, pale RBCs ⭐                │
│ • Ameboid trophozoites                 │
│ • Schüffner's dots ⭐ (RBC stippling)   │
│ • Oval gametocytes                     │
│                                        │
│ P. OVALE:                              │
│ • OVAL, fimbriated RBCs ⭐              │
│ • Schüffner's dots                     │
│ • Similar to P. vivax                  │
│                                        │
│ P. MALARIAE:                           │
│ • "Band forms" or "bar forms" ⭐        │
│ • Rosette schizonts (6-12 merozoites)  │
│ • Normal-sized RBCs                    │
│                                        │
│ P. KNOWLESI:                           │
│ • Resembles P. falciparum and P. malariae│
│ • All stages present                   │
│ • PCR for definitive diagnosis         │
└────────────────────────────────────────┘

RAPID DIAGNOSTIC TESTS (RDTs): ⭐⭐⭐⭐
┌────────────────────────────────────────┐
│ • Immunochromatographic tests          │
│ • Detect parasite ANTIGENS in blood    │
│                                        │
│ TARGETS:                               │
│ • HRP-2 (Histidine-Rich Protein 2):    │
│   - P. falciparum specific ⭐           │
│   - Persists after treatment (weeks)   │
│ • pLDH (Plasmodium Lactate             │
│   Dehydrogenase):                      │
│   - Pan-malarial or species-specific   │
│   - Clears faster after treatment      │
│                                        │
│ ADVANTAGES:                            │
│ • Rapid (15-20 minutes)                │
│ • No microscopy needed                 │
│ • Field use, resource-limited settings │
│ • High sensitivity for P. falciparum   │
│                                        │
│ LIMITATIONS:                           │
│ • Cannot quantify parasitemia          │
│ • Lower sensitivity at low parasitemia │
│ • False positives (HRP-2 persists)     │
│ • Cannot detect treatment failure early│
│ • Some gene deletions (hrp2/3 deletions)│
└────────────────────────────────────────┘

MOLECULAR:
• PCR: Most sensitive, speciation
• Not widely available, expensive
• Research, surveillance

SEROLOGY:
• IgG antibodies
• Past exposure, NOT acute diagnosis
• Epidemiologic surveys

OTHER LABS:
• Anemia (normocytic, hemolytic)
• Thrombocytopenia ⭐ (common, nonspecific)
• Elevated LDH, indirect bilirubin
• Elevated transaminases (mild)
• Hypoglycemia
• Lactic acidosis (severe)
```

## TREATMENT ⭐⭐⭐⭐⭐

### Uncomplicated Malaria

```
P. FALCIPARUM (or unknown species):
┌────────────────────────────────────────┐
│ ARTEMISININ-BASED COMBINATION          │
│ THERAPY (ACT): ⭐⭐⭐⭐⭐ (First-line)   │
│                                        │
│ Recommended ACTs:                      │
│ • Artemether-Lumefantrine (Coartem) ⭐  │
│   - 6 doses over 3 days                │
│   - With fatty food (↑ absorption)     │
│                                        │
│ • Artesunate-Amodiaquine              │
│ • Artesunate-Mefloquine               │
│ • Dihydroartemisinin-Piperaquine      │
│ • Artesunate-Pyronaridine             │
│                                        │
│ Rationale for ACT:                     │
│ • Artemisinin: Rapid parasite clearance│
│ • Partner drug: Eliminates residual    │
│ • Reduces resistance development       │
│                                        │
│ ALTERNATIVES (if ACT unavailable):     │
│ • Atovaquone-Proguanil (Malarone) ⭐    │
│   - 4 tablets daily × 3 days           │
│   - With food                          │
│   - Expensive                          │
│ • Quinine + Doxycycline/Clindamycin ⭐  │
│   - Quinine 650 mg PO TID × 3-7 days   │
│   - + Doxycycline 100 mg BID × 7 days  │
│   - Side effects: Cinchonism           │
│     (tinnitus, headache, nausea)       │
│ • Mefloquine (resistance increasing)   │
│                                        │
│ PREGNANCY: ⭐⭐⭐⭐                      │
│ • 1st trimester: Quinine + clindamycin │
│ • 2nd/3rd trimester: ACT (preferred) or│
│   quinine + clindamycin                │
│ • Atovaquone-proguanil: Limited data   │
│ • AVOID: Doxycycline, primaquine       │
└────────────────────────────────────────┘

P. VIVAX, P. OVALE, P. MALARIAE:
┌────────────────────────────────────────┐
│ BLOOD STAGE (Acute infection):         │
│ • Chloroquine 600 mg base (1000 mg     │
│   salt) × 1, then 300 mg (500 mg salt) │
│   at 6h, 24h, 48h                      │
│   Total: 1500 mg base over 3 days      │
│                                        │
│ • If chloroquine-resistant P. vivax:   │
│   Use ACT (same as P. falciparum)      │
│                                        │
│ LIVER STAGE (P. vivax, P. ovale): ⭐⭐⭐│
│ (Eradicate hypnozoites → prevent       │
│  relapses)                             │
│                                        │
│ PRIMAQUINE: ⭐⭐⭐⭐⭐                   │
│ • 30 mg base (52.6 mg salt) daily ×    │
│   14 days                              │
│ • OR 0.5 mg/kg/day × 14 days           │
│                                        │
│ BEFORE giving primaquine:              │
│ • CHECK G6PD STATUS ⭐⭐⭐⭐⭐           │
│ • Contraindicated if G6PD deficient    │
│   (causes hemolysis)                   │
│ • Alternative for G6PD deficiency:     │
│   Weekly primaquine × 8 weeks (under   │
│   supervision) or tafenoquine (single  │
│   dose, if available)                  │
│                                        │
│ • Also contraindicated: Pregnancy,     │
│   infants                              │
│                                        │
│ TAFENOQUINE (new): ⭐                   │
│ • Single dose (300 mg)                 │
│ • Radical cure                         │
│ • Requires G6PD testing (≥70% activity)│
│ • Ages ≥16 years                       │
└────────────────────────────────────────┘

P. MALARIAE:
• Chloroquine (same as P. vivax)
• NO hypnozoites, NO primaquine needed
```

### Severe Malaria ⭐⭐⭐⭐⭐

```
┌────────────────────────────────────────┐
│ MEDICAL EMERGENCY - ICU ADMISSION      │
│                                        │
│ ANTIMALARIAL: ⭐⭐⭐⭐⭐                 │
│                                        │
│ ARTESUNATE IV: ⭐⭐⭐⭐⭐ (First-line)   │
│ • 2.4 mg/kg IV at 0, 12, 24 hours      │
│ • Then daily until patient can take PO │
│ • Switch to ACT (complete 3 days total)│
│                                        │
│ SUPERIOR to quinine: ⭐                 │
│ • 23% reduction in mortality           │
│ • Faster parasite clearance            │
│ • Fewer side effects                   │
│ • Preferred in ALL age groups,         │
│   including pregnancy                  │
│                                        │
│ ALTERNATIVE (if artesunate unavailable):│
│ • Quinidine IV (US) or Quinine IV      │
│   - Loading dose, then continuous      │
│     infusion                           │
│   - Cardiac monitoring (QT prolongation)│
│   - Hypoglycemia risk                  │
│   - Switch to PO when able             │
│                                        │
│ SUPPORTIVE CARE: ⭐⭐⭐⭐                │
│ • Fluid management (careful! Risk of   │
│   pulmonary edema)                     │
│ • Treat hypoglycemia (frequent glucose │
│   monitoring)                          │
│ • Treat seizures (benzodiazepines)     │
│ • Transfusion if severe anemia (Hb <5) │
│ • Dialysis if AKI                      │
│ • Mechanical ventilation if ARDS       │
│ • Antibiotics if bacterial coinfection │
│                                        │
│ AVOID:                                 │
│ • Steroids (cerebral malaria) - harmful│
│ • Exchange transfusion (no proven      │
│   benefit, not recommended)            │
│ • Prophylactic anticonvulsants         │
└────────────────────────────────────────┘
```

### Monitoring & Follow-up
```
• Parasitemia: Daily until negative
• Glucose: Frequent (especially with quinine)
• Clinical improvement: 48-72 hours
• Treatment failure:
  - Persistent/rising parasitemia at 72h
  - Clinical deterioration
  - Consider resistance, wrong diagnosis
```

## PREVENTION ⭐⭐⭐⭐⭐

### Vector Control
```
┌────────────────────────────────────────┐
│ INSECTICIDE-TREATED NETS (ITNs): ⭐⭐⭐⭐│
│ • Long-lasting insecticidal nets (LLINs)│
│ • Most effective intervention          │
│ • Pyrethroid-impregnated               │
│ • Protects during sleep (night biting) │
│ • Community benefit (herd protection)  │
│                                        │
│ INDOOR RESIDUAL SPRAYING (IRS):        │
│ • Insecticides on interior walls       │
│ • Kills mosquitoes after feeding       │
│                                        │
│ ENVIRONMENTAL:                         │
│ • Eliminate breeding sites             │
│ • Drain stagnant water                 │
│ • Larvicides                           │
│                                        │
│ PERSONAL PROTECTION:                   │
│ • Repellents (DEET, picaridin)         │
│ • Protective clothing (long sleeves)   │
│ • Air conditioning, screens            │
│ • Avoid outdoor activities dusk-dawn   │
└────────────────────────────────────────┘
```

### Chemoprophylaxis ⭐⭐⭐⭐⭐

```
INDICATIONS:
• Travelers to endemic areas
• Expatriates in endemic areas
• Military personnel

CHOICE DEPENDS ON:
• Destination (resistance pattern)
• Duration of stay
• Contraindications, side effects
• Cost

REGIMENS:
┌────────────────────────────────────────┐
```

## HIGH-YIELD MNEMONICS

### 1. **Hypnozoites = "Sleeping Bugs"**
- **P. vivax** and **P. ovale** only
- Dormant liver forms → Relapses
- Need **Primaquine** to eradicate

### 2. **Fever Periodicity - "3-4-1"**
- **Tertian** (48h, every **3**rd day): P. falciparum, vivax, ovale
- **Quartan** (72h, every **4**th day): P. malariae
- **Quotidian** (24h, **1** day): P. knowlesi

### 3. **P. falciparum Features - "BHMR"**
- **B**anana-shaped gametocytes (crescent)
- **H**igh parasitemia (>5%)
- **M**ultiple rings per RBC
- **R**ings only (in peripheral blood)

### 4. **Severe Malaria - "CRACKS"**
- **C**erebral malaria
- **R**espiratory distress (ARDS)
- **A**cidosis (metabolic)
- **C**irculatory collapse (shock)
- **K**idney injury (acute)
- **S**evere anemia / Seizures

### 5. **G6PD Before Primaquine - "G Before P"**
- Check **G6PD** status before giving **Primaquine**
- Hemolysis if deficient

### 6. **ACT Components - "Art Partner"**
- **Art**emisinin (rapid kill)
- **Partner** drug (complete cure, prevent resistance)

### 7. **Anopheles Features - "45 Night Spot"**
- **45°** resting angle
- **Night** feeding (nocturnal)
- **Spot**ted wings

## NEET PG MCQS

### Q1. Which Plasmodium species forms hypnozoites in liver?
A) P. falciparum
B) P. vivax and P. ovale
C) P. malariae
D) All species

**Answer: B**
**Explanation**: Only P. vivax and P. ovale form dormant hypnozoites in liver → relapses. Require primaquine for radical cure.

---

### Q2. Banana-shaped gametocytes are seen in:
A) P. vivax
B) P. ovale
C) P. falciparum
D) P. malariae

**Answer: C**
**Explanation**: P. falciparum has characteristic crescent/banana-shaped gametocytes (pathognomonic).

---

### Q3. Quartan fever (72-hour periodicity) is caused by:
A) P. falciparum
B) P. vivax
C) P. malariae
D) P. ovale

**Answer: C**
**Explanation**: P. malariae has 72-hour cycle → fever every 4th day (quartan). Others have 48-hour (tertian).

---

### Q4. Before giving primaquine, must check:
A) Hemoglobin level
B) G6PD status
C) Liver function
D) Renal function

**Answer: B**
**Explanation**: Primaquine causes hemolysis in G6PD deficiency. Must check G6PD status before treatment.

---

### Q5. First-line treatment for uncomplicated P. falciparum malaria?
A) Chloroquine
B) Quinine alone
C) Artemisinin-based combination therapy (ACT)
D) Primaquine

**Answer: C**
**Explanation**: ACT (e.g., artemether-lumefantrine) is first-line for uncomplicated P. falciparum due to widespread chloroquine resistance.

---

### Q6. First-line treatment for severe malaria?
A) Oral chloroquine
B) IV artesunate
C) IV quinine
D) Oral ACT

**Answer: B**
**Explanation**: IV artesunate is first-line for severe malaria (23% mortality reduction vs quinine). Switch to oral ACT when able.

---

### Q7. Malaria transmitted by:
A) Aedes mosquito
B) Culex mosquito
C) Anopheles mosquito
D) Mansonia mosquito

**Answer: C**
**Explanation**: ONLY Anopheles mosquito transmits malaria. Female mosquito, nocturnal feeding, 45° resting angle.

---

### Q8. Definitive host of Plasmodium is:
A) Human
B) Anopheles mosquito
C) Both
D) Neither

**Answer: B**
**Explanation**: Anopheles mosquito is definitive host (sexual reproduction/sporogony). Humans are intermediate host (asexual reproduction/schizogony).

---

### Q9. Most dangerous malaria species?
A) P. vivax
B) P. ovale
C) P. falciparum
D) P. malariae

**Answer: C**
**Explanation**: P. falciparum causes most deaths (cerebral malaria, severe disease, can infect all RBCs → high parasitemia).

---

### Q10. Pregnant woman traveling to malaria-endemic area. Best chemoprophylaxis?
A) Doxycycline
B) Mefloquine
C) Primaquine
D) Atovaquone-proguanil

**Answer: B**
**Explanation**: Mefloquine is safe in all trimesters of pregnancy. Avoid doxycycline (pregnancy), primaquine (pregnancy), atovaquone-proguanil (limited data).

## CLINICAL PEARLS ⭐

1. **Only P. vivax and P. ovale have hypnozoites** → relapses → need primaquine
2. **P. falciparum is the killer** (cerebral malaria, severe disease)
3. **Banana-shaped gametocytes = P. falciparum** (pathognomonic)
4. **Check G6PD before primaquine** (hemolysis if deficient)
5. **Thick smear for detection**, **thin smear for species ID**
6. **ACT is first-line for P. falciparum** (chloroquine resistance widespread)
7. **IV artesunate for severe malaria** (superior to quinine)
8. **Thrombocytopenia common in malaria** (nonspecific but suggestive)
9. **Anopheles = only malaria vector** (female, nocturnal, 45° angle)
10. **Hypoglycemia in severe malaria** (especially with quinine) - monitor frequently
11. **Cerebral malaria**: Coma, seizures, high mortality (15-20%)
12. **Pregnancy increases malaria risk** (placental sequestration, low birth weight)
13. **Mefloquine safe in pregnancy** (for prophylaxis)
14. **Avoid steroids in cerebral malaria** (harmful, no benefit)
15. **ITNs most effective prevention** (long-lasting insecticidal nets)
16. **RBC morphology key to species ID**: Enlarged (vivax), oval/fimbriated (ovale), band forms (malariae)
17. **Quartan fever = malariae** (every 4th day, 72-hour cycle)
18. **P. malariae: No hypnozoites, no relapses** (but can persist decades)
19. **Hyperparasitemia >5% = severe disease risk** (unique to falciparum)
20. **Artemisinin resistance emerging** (Southeast Asia) - use with partner drug (ACT)
