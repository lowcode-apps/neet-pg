# HIV/AIDS

## Overview
Human Immunodeficiency Virus (HIV) is a retrovirus that causes Acquired Immunodeficiency Syndrome (AIDS), characterized by progressive depletion of CD4+ T cells leading to opportunistic infections and malignancies.

---

## 1. VIROLOGY ⭐⭐⭐⭐⭐

### Classification
- **Family**: Retroviridae
- **Genus**: Lentivirus
- **Types**:
  - **HIV-1**: Worldwide distribution, more virulent (most common)
  - **HIV-2**: West Africa, less virulent, slower progression

### Structure
```
HIV Virion Structure:
         ┌───────────────────────────┐
         │   Lipid Envelope          │
         │   (from host cell)        │
         │                           │
         │  ┌─────────────────────┐  │
         │  │ gp120 (attachment)  │  │ ← Surface glycoprotein
         │  │ gp41 (fusion)       │  │ ← Transmembrane protein
         │  └─────────────────────┘  │
         │                           │
         │  Matrix (p17)             │
         │  ┌─────────────────────┐  │
         │  │ Capsid (p24) ⭐      │  │ ← Core antigen
         │  │                     │  │
         │  │ • 2 copies RNA      │  │
         │  │ • Reverse           │  │
         │  │   Transcriptase     │  │
         │  │ • Integrase         │  │
         │  │ • Protease          │  │
         │  └─────────────────────┘  │
         └───────────────────────────┘

Key Proteins (remember for NEET PG):
┌────────────────────────────────────────┐
│ STRUCTURAL:                            │
│ • gp120/gp41 - Envelope glycoproteins  │
│ • p24 - Capsid (used in Ag tests)      │
│ • p17 - Matrix                         │
│                                        │
│ ENZYMES:                               │
│ • Reverse Transcriptase (RT) ⭐⭐⭐⭐⭐ │
│   RNA → DNA                            │
│ • Integrase - Integrates into host DNA │
│ • Protease - Cleaves viral polyproteins│
│                                        │
│ REGULATORY:                            │
│ • Tat - Transcription activator        │
│ • Rev - RNA export                     │
│ • Nef - Immune evasion                 │
└────────────────────────────────────────┘
```

### Genome
- **Two copies** of single-stranded **positive-sense RNA**
- **Diploid** (only diploid RNA virus)
- **9.7 kb** in length
- Contains **9 genes**: gag, pol, env + 6 regulatory genes

## 2. TRANSMISSION ⭐⭐⭐⭐⭐

### Routes of Transmission
```
┌────────────────────────────────────────┐
│ 1. SEXUAL (75-85% globally) ⭐         │
│    • Heterosexual (most common)        │
│    • MSM (men who have sex with men)   │
│    • Anal > Vaginal > Oral             │
│    • Risk factors: STIs, mucosal trauma│
│                                        │
│ 2. PARENTERAL (10-15%)                 │
│    • IV drug use (needle sharing)      │
│    • Blood transfusion (rare now)      │
│    • Needlestick injury (0.3% risk) ⭐ │
│    • Contaminated instruments          │
│                                        │
│ 3. VERTICAL (Mother-to-Child) (5-10%)  │
│    • Intrauterine (transplacental)     │
│    • Intrapartum (most common) ⭐      │
│    • Breastfeeding (10-15% additional) │
│    • Risk: 15-45% without intervention │
│    • Risk: <2% with ART ⭐⭐⭐⭐⭐       │
└────────────────────────────────────────┘
```

### NOT Transmitted By:
- Casual contact (hugging, shaking hands)
- Sharing utensils, toilets
- Insect bites (mosquitoes)
- Saliva, tears, sweat (insufficient viral load)

## 3. PATHOGENESIS ⭐⭐⭐⭐⭐

### Viral Entry and Replication
```
HIV Life Cycle (Drug Targets):
┌────────────────────────────────────────┐
│ 1. ATTACHMENT                          │
│    gp120 binds CD4 receptor            │
│    ↓                                   │
│ 2. CO-RECEPTOR BINDING                 │
│    CCR5 (macrophages) or CXCR4 (T cells)│
│    [Entry Inhibitors: Maraviroc]       │
│    ↓                                   │
│ 3. FUSION                              │
│    gp41 mediates membrane fusion       │
│    [Fusion Inhibitors: Enfuvirtide]    │
│    ↓                                   │
│ 4. REVERSE TRANSCRIPTION ⭐             │
│    RNA → DNA (reverse transcriptase)   │
│    [NRTIs, NNRTIs]                     │
│    ↓                                   │
│ 5. INTEGRATION                         │
│    Viral DNA integrates into host      │
│    [Integrase Inhibitors: Raltegravir] │
│    ↓                                   │
│ 6. TRANSCRIPTION & TRANSLATION         │
│    Host machinery produces viral       │
│    proteins                            │
│    ↓                                   │
│ 7. ASSEMBLY & BUDDING                  │
│    New virions form and bud            │
│    ↓                                   │
│ 8. MATURATION                          │
│    Protease cleaves polyproteins       │
│    [Protease Inhibitors]               │
└────────────────────────────────────────┘
```

## 4. CLINICAL STAGES ⭐⭐⭐⭐⭐

### Stage 1: Acute HIV Syndrome (Acute Retroviral Syndrome)
**Timing**: 2-4 weeks post-exposure
**Duration**: 1-2 weeks

```
Clinical Features (Mononucleosis-like):
┌────────────────────────────────────────┐
│ CONSTITUTIONAL (most common):          │
│ • Fever (80-90%)                       │
│ • Fatigue, malaise                     │
│ • Myalgias, arthralgias                │
│                                        │
│ ENT:                                   │
│ • Pharyngitis (50-70%)                 │
│ • Lymphadenopathy (generalized)        │
│                                        │
│ DERMATOLOGIC:                          │
│ • Maculopapular rash (40-80%) ⭐       │
│   Trunk and face, palm/soles spared    │
│ • Mucocutaneous ulcers                 │
│                                        │
│ GI:                                    │
│ • Nausea, vomiting, diarrhea           │
│                                        │
│ NEUROLOGIC:                            │
│ • Headache                             │
│ • Aseptic meningitis (rare)            │
└────────────────────────────────────────┘

Lab Findings:
• Very high viral load (>1 million copies/mL)
• CD4 count may drop transiently
• Atypical lymphocytosis
```

### Stage 2: Clinical Latency (Chronic HIV Infection)
**Duration**: 2-10+ years (varies widely)

- **Asymptomatic** or **persistent generalized lymphadenopathy (PGL)**
- **CD4 count**: Gradual decline (50-100 cells/μL per year without treatment)
- **Viral load**: Stable but detectable
- **Active viral replication** continues despite lack of symptoms
- **Treatment**: Highly recommended to prevent progression

### Stage 3: AIDS ⭐⭐⭐⭐⭐

**Definition** (ANY of the following):
1. **CD4 count <200 cells/μL** OR
2. **CD4 percentage <14%** OR
3. **AIDS-defining illness** (regardless of CD4 count)

```
AIDS-Defining Illnesses by CD4 Count:
┌────────────────────────────────────────┐
│ CD4 <500:                              │
│ • Tuberculosis (pulmonary)             │
│ • Herpes zoster (shingles)             │
│ • Bacterial pneumonia (recurrent)      │
│ • Oral/vaginal candidiasis             │
│                                        │
│ CD4 <200: ⭐⭐⭐⭐⭐                     │
│ • Pneumocystis jirovecii pneumonia     │
│   (PCP) - most common OI               │
│ • Disseminated histoplasmosis          │
│ • Candida esophagitis                  │
│                                        │
│ CD4 <100:                              │
│ • Toxoplasma encephalitis              │
│ • Cryptococcal meningitis              │
│ • Cryptosporidium diarrhea             │
│                                        │
│ CD4 <50:                               │
│ • CMV retinitis/colitis                │
│ • Mycobacterium avium complex (MAC)    │
│ • Progressive multifocal               │
│   leukoencephalopathy (PML - JC virus) │
└────────────────────────────────────────┘

Malignancies:
• Kaposi sarcoma (HHV-8)
• Non-Hodgkin lymphoma (especially CNS)
• Invasive cervical cancer
```

## 5. DIAGNOSIS ⭐⭐⭐⭐⭐

### Screening Algorithm (4th Generation)
```
Step 1: HIV-1/2 Antigen/Antibody Combo Test (4th Gen)
        ↓
     Positive?
        ↓
    ┌───┴───┐
   YES      NO → HIV Negative
    │             (if >4 weeks post-exposure)
    ↓
Step 2: HIV-1/HIV-2 Antibody Differentiation
        ↓
    ┌───┴───┐
   HIV-1  HIV-2
    │       │
    ↓       ↓
Step 3: HIV-1 Viral Load (RNA PCR)
        Confirms diagnosis
        Quantifies viremia
```

### Diagnostic Tests
```
┌────────────────────────────────────────┐
│ SCREENING (Initial):                   │
│                                        │
│ 4th Gen Combo Test (Preferred): ⭐⭐⭐⭐│
│ • Detects HIV-1/2 antibodies           │
│ • Detects p24 antigen                  │
│ • Window period: 2-4 weeks             │
│                                        │
│ 3rd Gen Antibody-only:                 │
│ • Window period: 3-6 weeks             │
│                                        │
│ Rapid Tests (Point-of-care):           │
│ • OraQuick (oral fluid/blood)          │
│ • Results in 20-40 minutes             │
│ • Sensitivity 99%, Specificity 99%     │
└────────────────────────────────────────┘

┌────────────────────────────────────────┐
│ CONFIRMATORY:                          │
│                                        │
│ HIV-1/2 Antibody Differentiation ⭐    │
│ • Western Blot (older method)          │
│ • Line immunoassay                     │
│                                        │
│ HIV RNA (Viral Load): ⭐⭐⭐⭐⭐         │
│ • Quantitative PCR                     │
│ • Confirms active infection            │
│ • Used for monitoring treatment        │
│ • Undetectable = <20-50 copies/mL      │
└────────────────────────────────────────┘

┌────────────────────────────────────────┐
│ SPECIAL SITUATIONS:                    │
│                                        │
│ Acute HIV (Window Period):             │
│ • HIV RNA (viral load) - detectable    │
│   7-10 days post-infection             │
│                                        │
│ Neonatal Testing: ⭐                    │
│ • HIV DNA PCR (not antibodies!)        │
│ • Maternal antibodies cross placenta   │
│ • Test at birth, 1-2 months, 4-6 months│
└────────────────────────────────────────┘
```

### CD4 Count and Viral Load Monitoring
```
Baseline Assessment:
┌────────────────────────────────────────┐
│ • CD4 count & percentage               │
│ • HIV viral load                       │
│ • CBC, CMP, LFTs                       │
│ • Lipid panel                          │
│ • HBV, HCV, syphilis screening         │
│ • PPD or IGRA (TB screening)           │
│ • Toxoplasma IgG                       │
│ • G6PD (if TMP-SMX planned)            │
│ • Pregnancy test (women)               │
│ • HLA-B*5701 (before abacavir) ⭐      │
└────────────────────────────────────────┘

Monitoring on ART:
• Viral load: 2-8 weeks after starting, then every 3-6 months
• CD4 count: Every 3-6 months
• Goal: Undetectable viral load (<50 copies/mL)
```

## 6. TREATMENT ⭐⭐⭐⭐⭐

### Antiretroviral Therapy (ART)

#### When to Start
```
UNIVERSAL RECOMMENDATION: ⭐⭐⭐⭐⭐
┌────────────────────────────────────────┐
│ Start ART in ALL HIV+ patients         │
│ regardless of CD4 count                │
│                                        │
│ U=U (Undetectable = Untransmittable)   │
│ • Suppressed viral load prevents       │
│   sexual transmission                  │
└────────────────────────────────────────┘

URGENT (Start immediately):
• Pregnancy
• AIDS-defining illness
• Acute opportunistic infection
• CD4 <200 cells/μL
• HIV-associated nephropathy
• Acute HIV infection (consider)
```

#### ART Classes and Mechanisms
```
┌────────────────────────────────────────┐
│ 1. NRTIs (Nucleoside RTIs) ⭐⭐⭐⭐⭐    │
│    "Backbone" of regimens              │
│                                        │
│    • Tenofovir (TDF or TAF)            │
│      - Nephrotoxic (TDF), osteoporosis │
│      - TAF preferred (less toxic)      │
│                                        │
│    • Emtricitabine (FTC)               │
│      - Well tolerated                  │
│                                        │
│    • Abacavir (ABC)                    │
│      - Check HLA-B*5701 first ⭐       │
│      - Fatal hypersensitivity if +     │
│      - Contraindicated if CVD risk     │
│                                        │
│    • Lamivudine (3TC)                  │
│      - Also active against HBV         │
│                                        │
│    • Zidovudine (AZT)                  │
│      - Bone marrow suppression         │
│      - Used in pregnancy (PMTCT)       │
│                                        │
│    Adverse Effects:                    │
│    - Lactic acidosis (rare, all)       │
│    - Lipoatrophy (stavudine, zidovudine)│
└────────────────────────────────────────┘

┌────────────────────────────────────────┐
│ 2. NNRTIs (Non-Nucleoside RTIs)        │
│                                        │
│    • Efavirenz (EFV)                   │
│      - CNS effects (vivid dreams,      │
│        dizziness) ⭐                    │
│      - Teratogenic (avoid in pregnancy)│
│      - CYP inducer                     │
│                                        │
│    • Nevirapine (NVP)                  │
│      - Hepatotoxic, rash               │
│      - Used in PMTCT                   │
│                                        │
│    • Rilpivirine (RPV)                 │
│      - Better tolerated than EFV       │
│      - Requires food, avoid PPIs       │
│                                        │
│    • Doravirine (newer)                │
│                                        │
│    Adverse Effects:                    │
│    - Rash (especially nevirapine)      │
│    - Hepatotoxicity                    │
│    - Low genetic barrier to resistance │
└────────────────────────────────────────┘

┌────────────────────────────────────────┐
│ 3. PIs (Protease Inhibitors)           │
│    Always boosted with ritonavir/      │
│    cobicistat                          │
│                                        │
│    • Darunavir/r (DRV/r)               │
│      - High genetic barrier            │
│                                        │
│    • Atazanavir/r (ATV/r)              │
│      - Hyperbilirubinemia (indirect) ⭐│
│      - Jaundice (benign)               │
│      - Nephrolithiasis                 │
│                                        │
│    • Lopinavir/ritonavir (LPV/r)       │
│      - GI side effects                 │
│      - Safe in pregnancy               │
│                                        │
│    Adverse Effects (Class):            │
│    - GI upset (diarrhea, nausea)       │
│    - Metabolic: ↑ lipids, ↑ glucose,   │
│      lipodystrophy                     │
│    - Drug interactions (CYP3A4)        │
└────────────────────────────────────────┘

┌────────────────────────────────────────┐
│ 4. INSTIs (Integrase Inhibitors) ⭐⭐⭐⭐│
│    "Preferred" class - best tolerated  │
│                                        │
│    • Dolutegravir (DTG) ⭐⭐⭐⭐⭐       │
│      - High barrier to resistance      │
│      - Once daily                      │
│      - Well tolerated                  │
│      - Insomnia, weight gain           │
│      - Neural tube defects (early data,│
│        likely safe now)                │
│                                        │
│    • Bictegravir (BIC)                 │
│      - Similar to DTG                  │
│      - Part of Biktarvy (BIC/TAF/FTC)  │
│                                        │
│    • Raltegravir (RAL)                 │
│      - Twice daily                     │
│      - Low genetic barrier             │
│                                        │
│    • Elvitegravir/c (EVG/c)            │
│      - Requires boosting               │
│                                        │
│    Adverse Effects:                    │
│    - Generally minimal                 │
│    - Weight gain                       │
│    - Insomnia, headache                │
└────────────────────────────────────────┘

┌────────────────────────────────────────┐
│ 5. Entry/Fusion Inhibitors             │
│                                        │
│    • Maraviroc (MVC)                   │
│      - CCR5 antagonist                 │
│      - Requires tropism testing        │
│                                        │
│    • Enfuvirtide (T-20)                │
│      - Fusion inhibitor                │
│      - SC injection (rarely used)      │
└────────────────────────────────────────┘
```

#### Recommended First-Line Regimens (2024)
```
┌────────────────────────────────────────┐
│ PREFERRED (INSTI-based): ⭐⭐⭐⭐⭐      │
│                                        │
│ Biktarvy (BIC/TAF/FTC)                 │
│ • Single tablet, once daily            │
│ • Best tolerated                       │
│                                        │
│ Dovato (DTG/3TC)                       │
│ • 2-drug regimen                       │
│ • If VL <500K, no HBV, no resistance   │
│                                        │
│ DTG + (TAF or TDF) + (FTC or 3TC)      │
│ • Generic option                       │
└────────────────────────────────────────┘

Standard Regimen Structure:
2 NRTIs + (INSTI or NNRTI or PI/r)
     ↑           ↑
  "Backbone"  "Anchor"
```

### Treatment Monitoring
```
Goals of ART:
┌────────────────────────────────────────┐
│ 1. VIROLOGIC:                          │
│    • Undetectable VL (<50 copies/mL)   │
│      within 6 months ⭐                 │
│    • Maintain suppression              │
│                                        │
│ 2. IMMUNOLOGIC:                        │
│    • CD4 count increase                │
│    • Immune reconstitution             │
│                                        │
│ 3. CLINICAL:                           │
│    • Prevent opportunistic infections  │
│    • Prevent transmission (U=U)        │
│    • Improve quality of life           │
└────────────────────────────────────────┘

Virologic Failure:
• VL >200 copies/mL after 24 weeks on ART
• VL detectable after prior suppression
• Consider: adherence, drug interactions, resistance
• Genotypic resistance testing
```

## 8. PREVENTION ⭐⭐⭐⭐⭐

### Pre-Exposure Prophylaxis (PrEP)
```
Indications:
┌────────────────────────────────────────┐
│ HIV-negative individuals at high risk: │
│                                        │
│ • MSM with condomless sex              │
│ • Heterosexual with HIV+ partner       │
│ • IV drug users                        │
│ • Sex workers                          │
│ • Inconsistent condom use              │
└────────────────────────────────────────┘

Regimens:
┌────────────────────────────────────────┐
│ DAILY ORAL (Preferred): ⭐⭐⭐⭐⭐       │
│ • TDF/FTC (Truvada) 1 tab daily        │
│ • TAF/FTC (Descovy) 1 tab daily        │
│   (not for receptive vaginal sex)      │
│                                        │
│ Efficacy: >99% if adherent             │
│                                        │
│ ON-DEMAND (MSM only):                  │
│ • "2-1-1" dosing                       │
│ • 2 pills 2-24h before sex             │
│ • 1 pill 24h after first dose          │
│ • 1 pill 48h after first dose          │
│                                        │
│ INJECTABLE (NEW):                      │
│ • Cabotegravir LA (Apretude)           │
│ • Every 2 months                       │
│ • Superior to daily TDF/FTC            │
└────────────────────────────────────────┘

Monitoring:
• HIV test before starting (must be negative)
• HIV test every 3 months while on PrEP
• Creatinine, HBV/HCV screening
• STI screening every 3-6 months
```

### Post-Exposure Prophylaxis (PEP) ⭐⭐⭐⭐⭐
```
Indications:
┌────────────────────────────────────────┐
│ OCCUPATIONAL:                          │
│ • Needlestick/sharp injury with        │
│   HIV+ source                          │
│ • Mucous membrane/non-intact skin      │
│   exposure to HIV+ blood               │
│                                        │
│ NON-OCCUPATIONAL:                      │
│ • Condomless sex with HIV+ partner     │
│   (or unknown status)                  │
│ • Sexual assault                       │
│ • Needle sharing                       │
└────────────────────────────────────────┘

Timing: ⭐⭐⭐⭐⭐
• Start ASAP (ideally <2 hours)
• Up to 72 hours post-exposure
• NOT recommended after 72 hours

Regimen:
┌────────────────────────────────────────┐
│ PREFERRED (28 days):                   │
│                                        │
```

## 9. COMPLICATIONS AND SPECIAL SITUATIONS

### Immune Reconstitution Inflammatory Syndrome (IRIS) ⭐⭐⭐⭐
```
Definition:
Paradoxical worsening of symptoms after
starting ART due to restoration of immune
function against existing infections

Timing:
• Usually 2-12 weeks after ART initiation
• Rising CD4, falling VL

Common Presentations:
┌────────────────────────────────────────┐
│ • TB (most common) - worsening symptoms│
│   lymphadenopathy, fever               │
│ • Cryptococcal meningitis - increased  │
│   ICP, meningismus                     │
│ • CMV retinitis - vitreitis, vision loss│
│ • Kaposi sarcoma - new/enlarging lesions│
│ • PML - worsening neurologic deficits  │
│ • Hepatitis B or C - hepatitis flare   │
└────────────────────────────────────────┘

Management:
• Continue ART (usually)
• Treat underlying infection
• Corticosteroids for severe cases
• NSAIDs for mild symptoms

Prevention:
• Treat OIs before starting ART if possible
• Screen for subclinical infections
• Delay ART in cryptococcal meningitis
  (2-5 weeks after antifungal initiation)
```

### Drug Resistance
```
Causes:
• Poor adherence (most common)
• Suboptimal regimen
• Drug interactions
• Transmitted resistance (5-15%)

Testing:
• Genotypic resistance testing (preferred)
  - Before starting ART (if VL >500)
  - At virologic failure
• Phenotypic testing (if needed)

Resistance Patterns:
┌────────────────────────────────────────┐
│ NRTIs:                                 │
│ • M184V (FTC/3TC) - most common        │
│ • K65R, K70E (tenofovir)               │
│ • TAMs (thymidine analogue mutations)  │
│                                        │
│ NNRTIs:                                │
│ • Low genetic barrier                  │
│ • Single mutation → high-level resistance│
│ • K103N, Y181C, G190A                  │
│                                        │
│ PIs:                                   │
│ • High genetic barrier                 │
│ • Multiple mutations needed            │
│                                        │
│ INSTIs:                                │
│ • DTG/BIC: Very high barrier           │
│ • RAL/EVG: Lower barrier               │
└────────────────────────────────────────┘
```

## HIGH-YIELD MNEMONICS

### 1. **Acute HIV Syndrome (FLAPPERS)**
- **F**ever
- **L**ymphadenopathy
- **A**rthralgias/myalgias
- **P**haryngitis
- **P**atients have a
- **E**xanthem (rash)
- **R**eal bad
- **S**ickness

### 2. **AIDS-Defining CD4 Counts (2-1-0.5)**
- **<200**: PCP, Candida esophagitis
- **<100**: Toxoplasma, Cryptococcus
- **<50**: CMV, MAC

### 3. **PrEP Monitoring (HITS)**
- **H**IV test (every 3 months)
- **I**nfections (STIs every 3-6 months)
- **T**ruvada (adherence assessment)
- **S**erum creatinine (renal function)

### 4. **PEP Timing (2-72 Rule)**
- Best within **2 hours**
- Up to **72 hours**

### 5. **PMTCT Goals (3 A's)**
- **A**RT for mother (suppress VL)
- **A**void high-risk transmission (C-section if VL >1000)
- **A**ZT for baby (prophylaxis)

## NEET PG MCQS

### Q1. A patient presents with fever, pharyngitis, maculopapular rash, and lymphadenopathy 3 weeks after unprotected sex. HIV antibody test is negative. Next best test?
A) Repeat antibody test in 3 months
B) CD4 count
C) HIV RNA (viral load)
D) Western blot

**Answer: C**
**Explanation**: Acute HIV syndrome during window period. HIV RNA detectable before antibodies develop (7-10 days vs 3-6 weeks).

---

### Q2. An HIV patient has CD4 count of 150 cells/μL. Which prophylaxis is indicated?
A) TMP-SMX only
B) Azithromycin only
C) TMP-SMX + Azithromycin
D) No prophylaxis needed

**Answer: A**
**Explanation**: CD4 <200 → PCP prophylaxis (TMP-SMX). MAC prophylaxis only if CD4 <50. TMP-SMX also covers toxoplasmosis if IgG+.

---

### Q3. Best test to diagnose HIV in a 2-month-old infant born to HIV+ mother?
A) HIV antibody test
B) HIV DNA PCR
C) CD4 count
D) Viral load (HIV RNA)

**Answer: B**
**Explanation**: HIV DNA PCR for infants <18 months (maternal antibodies cross placenta, making antibody tests unreliable). HIV RNA also acceptable.

---

### Q4. An HIV patient on tenofovir/emtricitabine/efavirenz has persistent VL >1000 after 6 months. Most likely cause?
A) Drug resistance
B) Poor adherence
C) Inadequate dosing
D) Drug interaction

**Answer: B**
**Explanation**: Poor adherence is the most common cause of virologic failure. Assess adherence before assuming resistance.

---

### Q5. Healthcare worker sustains needlestick from HIV+ patient. PEP should be started within:
A) 2 hours (optimal)
B) 24 hours
C) 72 hours (maximum)
D) 7 days

**Answer: A (optimal), but C (maximum)**
**Explanation**: PEP most effective if started <2 hours, but can be started up to 72 hours post-exposure. NEET PG typically asks for "maximum" = 72 hours.

---

### Q6. Preferred first-line ART regimen currently is:
A) 2 NRTIs + NNRTI
B) 2 NRTIs + PI/r
C) 2 NRTIs + INSTI
D) NRTI-sparing regimen

**Answer: C**
**Explanation**: INSTI-based regimens (e.g., dolutegravir, bictegravir) are preferred due to efficacy, tolerability, and high genetic barrier.

---

### Q7. An HIV patient on ART develops worsening TB symptoms 3 weeks after starting treatment despite good adherence. CD4 increased from 50 to 150. Diagnosis?
A) Drug-resistant TB
B) Treatment failure
C) IRIS (Immune Reconstitution Inflammatory Syndrome)
D) New opportunistic infection

**Answer: C**
**Explanation**: IRIS - paradoxical worsening of OI symptoms 2-12 weeks after ART due to immune recovery, despite rising CD4 and falling VL.

---

### Q8. Which drug requires HLA-B*5701 testing before initiation?
A) Tenofovir
B) Abacavir
C) Efavirenz
D) Atazanavir

**Answer: B**
**Explanation**: Abacavir hypersensitivity reaction (potentially fatal) in HLA-B*5701+ patients. Test before starting.

## REVISION CHECKLIST

### Must Know for NEET PG:
- [ ] HIV structure: p24 capsid antigen, gp120/gp41 envelope, reverse transcriptase
- [ ] HIV life cycle: attachment (CD4/CCR5/CXCR4) → fusion → reverse transcription → integration → maturation
- [ ] Transmission: Sexual (most common), parenteral (0.3% needlestick), vertical (intrapartum > intrauterine)
- [ ] U=U: Undetectable viral load = Untransmittable
- [ ] Acute HIV syndrome: Mononucleosis-like, window period, very high VL, HIV RNA positive before antibodies
- [ ] AIDS definition: CD4 <200 OR AIDS-defining illness
- [ ] AIDS OIs by CD4: <200 (PCP), <100 (Toxo, Crypto), <50 (CMV, MAC)
- [ ] Diagnosis: 4th gen (Ag/Ab combo) → differentiation → viral load confirmation
- [ ] Infant testing: HIV DNA PCR (not antibodies - maternal antibodies cross placenta)
- [ ] ART: Start in ALL patients regardless of CD4
- [ ] First-line ART: 2 NRTIs + INSTI (e.g., Biktarvy = BIC/TAF/FTC)
- [ ] Treatment goal: Undetectable VL (<50) within 6 months
- [ ] NRTI toxicities: Tenofovir (nephrotoxic), Abacavir (hypersensitivity - check HLA-B*5701)
- [ ] NNRTI toxicities: Efavirenz (CNS effects, teratogenic)
- [ ] PI toxicities: GI upset, metabolic (lipids, glucose), Atazanavir (hyperbilirubinemia)
- [ ] INSTI: Best tolerated, preferred class, DTG/BIC
- [ ] PCP prophylaxis: TMP-SMX if CD4 <200
- [ ] MAC prophylaxis: Azithromycin if CD4 <50
- [ ] Toxoplasma prophylaxis: TMP-SMX if CD4 <100 AND IgG+
- [ ] PrEP: TDF/FTC daily for high-risk HIV-negative individuals, >99% effective if adherent
- [ ] PEP: Start <72 hours (ideally <2 hours), 28-day regimen (TDF/FTC + RAL/DTG)
- [ ] PMTCT: ART for mother (goal: undetectable VL), C-section if VL >1000, infant ZDV prophylaxis
- [ ] IRIS: Paradoxical worsening of OI symptoms 2-12 weeks after ART (rising CD4, falling VL)
- [ ] Drug resistance: Poor adherence most common cause, genotypic testing at failure

## CLINICAL PEARLS ⭐

1. **All HIV+ patients should start ART immediately** regardless of CD4 count
2. **Window period**: HIV RNA detectable before antibodies (use RNA for acute HIV)
3. **Maternal antibodies cross placenta**: Use HIV DNA PCR in infants, not antibody tests
4. **AIDS definition**: CD4 <200 OR AIDS-defining illness (regardless of CD4)
5. **PCP is most common OI**: Prophylaxis with TMP-SMX if CD4 <200
6. **TMP-SMX covers both PCP and Toxoplasma** (if IgG+)
7. **U=U (Undetectable = Untransmittable)**: Suppressed VL prevents sexual transmission
8. **INSTI-based regimens are preferred** (DTG, BIC) - best efficacy and tolerability
9. **Abacavir requires HLA-B*5701 testing** before use (fatal hypersensitivity if positive)
10. **Efavirenz causes CNS side effects** (vivid dreams, dizziness) and is teratogenic
11. **Atazanavir causes indirect hyperbilirubinemia** (benign jaundice)
12. **IRIS**: Paradoxical worsening after ART - continue ART and treat underlying OI
13. **PEP must start <72 hours** (ideally <2 hours) post-exposure
14. **PrEP is >99% effective** if adherent (daily TDF/FTC)
15. **PMTCT**: Maternal ART + infant ZDV → transmission <2%
16. **C-section if VL >1000** at delivery to reduce vertical transmission
17. **Poor adherence** is most common cause of treatment failure
18. **Genotypic resistance testing** before starting ART (if VL >500) and at failure
19. **CD4 count recovers slowly** (50-100 cells/year), but VL should be undetectable in 6 months
20. **Screen all pregnant women** for HIV (opt-out testing recommended)
