# Respiratory Viruses

## Overview
Respiratory viruses are major causes of upper and lower respiratory tract infections worldwide. This chapter covers influenza, respiratory syncytial virus (RSV), parainfluenza, adenovirus, rhinovirus, and coronavirus.

---

## 1. INFLUENZA VIRUSES ⭐⭐⭐⭐⭐

### Virology
```
CLASSIFICATION:
┌────────────────────────────────────────┐
│ Family: Orthomyxoviridae               │
│                                        │
│ TYPES (based on nucleoprotein):       │
│ • Influenza A ⭐⭐⭐⭐⭐                 │
│   - Humans, birds, pigs, horses        │
│   - Pandemics, epidemics, seasonal     │
│   - Most important                     │
│                                        │
│ • Influenza B                          │
│   - Humans only                        │
│   - Epidemics, seasonal                │
│   - Milder than A                      │
│                                        │
│ • Influenza C                          │
│   - Humans                             │
│   - Mild, sporadic                     │
│   - Less clinically significant        │
│                                        │
│ • Influenza D                          │
│   - Cattle (not humans)                │
└────────────────────────────────────────┘

STRUCTURE: ⭐⭐⭐⭐⭐
┌────────────────────────────────────────┐
│ • ENVELOPED virus                      │
│ • SEGMENTED genome (8 segments) ⭐      │
│   - Negative-sense ssRNA               │
│   - Allows REASSORTMENT (antigenic     │
│     shift)                             │
│                                        │
│ SURFACE GLYCOPROTEINS: ⭐⭐⭐⭐⭐        │
│                                        │
│ HEMAGGLUTININ (HA): ⭐⭐⭐⭐⭐           │
│ • Viral attachment to sialic acid      │
│ • Fusion and entry                     │
│ • 18 subtypes (H1-H18)                 │
│ • Target of neutralizing antibodies    │
│ • "H" in H1N1, H3N2, etc.              │
│                                        │
│ NEURAMINIDASE (NA): ⭐⭐⭐⭐⭐           │
│ • Cleaves sialic acid                  │
│ • Releases progeny virions             │
│ • 11 subtypes (N1-N11)                 │
│ • Target of antivirals (oseltamivir)   │
│ • "N" in H1N1, H3N2, etc.              │
│                                        │
│ M2 PROTEIN:                            │
│ • Ion channel                          │
│ • Target of amantadine (no longer used)│
└────────────────────────────────────────┘
```

### Antigenic Variation ⭐⭐⭐⭐⭐

```
TWO MECHANISMS:
┌────────────────────────────────────────┐
│ 1. ANTIGENIC DRIFT: ⭐⭐⭐⭐⭐           │
│    (Minor changes)                     │
│                                        │
│ Mechanism:                             │
│ • Point mutations in HA/NA genes       │
│ • RNA polymerase lacks proofreading    │
│ • Gradual accumulation of mutations    │
│                                        │
│ Result:                                │
│ • Minor changes in HA/NA epitopes      │
│ • Occurs CONTINUOUSLY                  │
│ • Partial immunity from previous       │
│   infection/vaccination                │
│                                        │
│ Clinical:                              │
│ • SEASONAL EPIDEMICS ⭐                 │
│ • Both Influenza A and B               │
│ • Vaccine updated annually             │
│                                        │
│ ┌──────────────────────────────────┐  │
│ │ 2. ANTIGENIC SHIFT: ⭐⭐⭐⭐⭐       │  │
│ │    (Major changes)                │  │
│ │                                   │  │
│ │ Mechanism:                        │  │
│ │ • REASSORTMENT of gene segments ⭐ │  │
│ │ • Co-infection with 2 different   │  │
│ │   influenza A strains             │  │
│ │ • Exchange of HA/NA segments      │  │
│ │ • Segmented genome allows this    │  │
│ │                                   │  │
│ │ "Mixing vessel":                  │  │
│ │ • Pigs ⭐ (have receptors for both │  │
│ │   human and avian influenza)      │  │
│ │ • Birds → Pigs → Humans           │  │
│ │                                   │  │
│ │ Result:                           │  │
│ │ • NEW HA/NA combination ⭐         │  │
│ │ • Completely novel virus          │  │
│ │ • No pre-existing immunity        │  │
│ │                                   │  │
│ │ Clinical:                         │  │
│ │ • PANDEMICS ⭐⭐⭐⭐⭐              │  │
│ │ • Influenza A ONLY (reassortment) │  │
│ │ • High morbidity/mortality        │  │
│ │ • Rapid global spread             │  │
│ │                                   │  │
│ │ Historical examples:              │  │
│ │ • 1918: H1N1 "Spanish flu"        │  │
│ │   (~50 million deaths)            │  │
│ │ • 1957: H2N2 "Asian flu"          │  │
│ │ • 1968: H3N2 "Hong Kong flu"      │  │
│ │ • 2009: H1N1 "Swine flu"          │  │
│ └──────────────────────────────────┘  │
└────────────────────────────────────────┘

CURRENT CIRCULATING STRAINS (seasonal):
• Influenza A: H1N1, H3N2
• Influenza B: Yamagata, Victoria lineages
```

### Transmission & Epidemiology
```
TRANSMISSION: ⭐⭐⭐⭐
• Respiratory droplets (coughing, sneezing)
• Direct contact
• Fomites (surfaces)
• Highly contagious

CONTAGIOUS PERIOD:
• 1 day before symptoms to 5-7 days after
• Children/immunocompromised: Longer

SEASONALITY:
• Temperate zones: Winter (Oct-May) ⭐
• Tropics: Year-round
• Southern hemisphere: Jun-Sep

INCUBATION: 1-4 days (average 2 days)
```

### Clinical Manifestations ⭐⭐⭐⭐⭐

```
UNCOMPLICATED INFLUENZA:
┌────────────────────────────────────────┐
│ SYSTEMIC (predominant): ⭐              │
│ • Abrupt onset HIGH FEVER (38-40°C) ⭐  │
│ • Chills, rigors                       │
│ • Severe MYALGIAS ⭐⭐⭐⭐               │
│ • Severe HEADACHE ⭐                    │
│ • Prostration, malaise                 │
│ • Anorexia                             │
│                                        │
│ RESPIRATORY (develop later):           │
│ • Dry cough (prominent, persistent) ⭐  │
│ • Sore throat                          │
│ • Rhinorrhea                           │
│ • Nasal congestion                     │
│                                        │
│ DURATION:                              │
│ • Acute illness: 3-7 days              │
│ • Cough/fatigue: Can persist weeks     │
│                                        │
│ KEY FEATURES:                          │
│ • ABRUPT onset ⭐                       │
│ • SYSTEMIC > respiratory (initially)   │
│ • High fever + myalgias hallmark       │
└────────────────────────────────────────┘

COMPLICATIONS: ⭐⭐⭐⭐⭐
┌────────────────────────────────────────┐
│ PRIMARY INFLUENZA PNEUMONIA: ⭐⭐⭐⭐    │
│ • Direct viral pneumonitis             │
│ • Rapid progression (24-48h)           │
│ • Bilateral interstitial infiltrates   │
│ • Hypoxemia, ARDS                      │
│ • High mortality                       │
│ • Risk: Cardiopulmonary disease,       │
│   pregnancy, immunocompromised         │
│                                        │
│ SECONDARY BACTERIAL PNEUMONIA: ⭐⭐⭐⭐  │
│ • 2-14 days after influenza            │
│ • Improvement then sudden worsening ⭐  │
│ • Organisms:                           │
│   - S. pneumoniae (most common) ⭐      │
│   - S. aureus (including MRSA)         │
│   - H. influenzae                      │
│   - Group A Streptococcus              │
│ • Purulent sputum, lobar consolidation │
│ • Treat with antibiotics               │
│                                        │
│ OTHER:                                 │
│ • Myositis (especially children)       │
│ • Myocarditis, pericarditis            │
│ • Encephalopathy (children)            │
│ • Rhabdomyolysis                       │
│ • Reye syndrome (aspirin use) ⭐        │
│   - NEVER give aspirin for influenza!  │
│ • Exacerbation of chronic diseases     │
│   (asthma, COPD, CHF)                  │
└────────────────────────────────────────┘

HIGH-RISK GROUPS:
• Age <5 years (especially <2)
• Age ≥65 years
• Pregnancy
• Chronic medical conditions (cardiac,
  pulmonary, renal, hepatic, DM, obesity)
• Immunosuppressed
• Nursing home residents
```

### Diagnosis ⭐⭐⭐⭐
```
┌────────────────────────────────────────┐
│ CLINICAL:                              │
│ • Often sufficient during flu season   │
│ • Classic presentation in epidemic     │
│                                        │
│ RAPID ANTIGEN TESTS: ⭐                 │
│ • Nasopharyngeal swab                  │
│ • Results in 15-30 minutes             │
│ • Moderate sensitivity (50-70%)        │
│ • High specificity (90-95%)            │
│ • Negative doesn't rule out            │
│ • Useful for quick decisions           │
│                                        │
│ RT-PCR: ⭐⭐⭐⭐⭐ (Gold standard)        │
│ • Nasopharyngeal swab                  │
│ • Most sensitive/specific              │
│ • Detects and types influenza          │
│ • Can test for multiple respiratory    │
│   viruses (multiplex)                  │
│ • Results in hours                     │
│                                        │
│ VIRAL CULTURE:                         │
│ • Slow (3-10 days)                     │
│ • Not for acute management             │
│ • Surveillance, research               │
│                                        │
│ SEROLOGY:                              │
│ • Retrospective (4-fold rise in titer) │
│ • Not useful acutely                   │
└────────────────────────────────────────┘
```

### Treatment ⭐⭐⭐⭐⭐

```
ANTIVIRALS:
┌────────────────────────────────────────┐
│ NEURAMINIDASE INHIBITORS: ⭐⭐⭐⭐⭐     │
│                                        │
│ OSELTAMIVIR (Tamiflu): ⭐⭐⭐⭐⭐        │
│ • Oral                                 │
│ • Treatment: 75 mg PO BID × 5 days     │
│ • Prophylaxis: 75 mg PO daily          │
│ • Pregnancy: SAFE, preferred ⭐         │
│ • Available for all ages (liquid)      │
│                                        │
│ ZANAMIVIR (Relenza):                   │
│ • Inhaled                              │
│ • 10 mg (2 inhalations) BID × 5 days   │
│ • Avoid in asthma/COPD (bronchospasm)  │
│                                        │
│ PERAMIVIR:                             │
│ • IV (single dose)                     │
│ • 600 mg IV × 1                        │
│ • For hospitalized patients            │
│                                        │
│ BALOXAVIR (Xofluza):                   │
│ • Oral, single dose                    │
│ • Cap-dependent endonuclease inhibitor │
│ • 40 mg (or 80 mg if ≥80 kg) × 1 dose  │
│ • Ages ≥12 years                       │
│                                        │
│ TIMING: ⭐⭐⭐⭐⭐                        │
│ • Most effective if started within     │
│   48 HOURS of symptom onset ⭐          │
│ • Reduces duration by ~1 day           │
│ • Reduces complications                │
│ • Later start still beneficial if      │
│   hospitalized or high-risk            │
│                                        │
│ INDICATIONS FOR TREATMENT: ⭐⭐⭐⭐      │
│ • ALL hospitalized patients            │
│ • Severe/progressive disease           │
│ • High-risk patients (even if >48h)    │
│ • Outpatients: If within 48h and       │
│   high-risk OR desire symptom reduction│
│                                        │
│ RESISTANCE:                            │
│ • Low for oseltamivir (sporadic)       │
│ • Baloxavir: Emerging resistance       │
└────────────────────────────────────────┘

OLDER ANTIVIRALS (Not Recommended):
┌────────────────────────────────────────┐
│ M2 INHIBITORS:                         │
│ • Amantadine, Rimantadine              │
│ • Influenza A only                     │
│ • HIGH RESISTANCE (>99%) ⭐             │
│ • No longer recommended                │
└────────────────────────────────────────┘

SUPPORTIVE CARE:
• Rest, hydration
• Antipyretics (acetaminophen, ibuprofen)
• AVOID aspirin in children (Reye syndrome)
• Symptomatic relief (cough suppressants)
```

### Prevention ⭐⭐⭐⭐⭐

```
VACCINATION: ⭐⭐⭐⭐⭐
┌────────────────────────────────────────┐
│ TYPES OF VACCINES:                     │
│                                        │
│ INACTIVATED INFLUENZA VACCINE (IIV): ⭐ │
│ • Intramuscular injection              │
│ • Trivalent (3 strains) or             │
│   Quadrivalent (4 strains)             │
│ • Standard-dose or High-dose (≥65)     │
│ • Adjuvanted (≥65)                     │
│ • Recombinant (egg-free)               │
│ • Age ≥6 months                        │
│                                        │
│ LIVE ATTENUATED INFLUENZA VACCINE      │
│ (LAIV): ⭐                              │
│ • Intranasal spray (FluMist)           │
│ • Quadrivalent                         │
│ • Ages 2-49 years                      │
│ • Contraindications:                   │
│   - Age <2 or ≥50                      │
│   - Pregnancy                          │
│   - Immunosuppression                  │
│   - Severe asthma                      │
│   - Aspirin therapy (children)         │
│                                        │
│ RECOMMENDATIONS: ⭐⭐⭐⭐⭐              │
│ • ANNUAL vaccination for ALL ≥6 months │
│ • Timing: Before flu season starts     │
│   (September-October)                  │
│ • Vaccine composition updated yearly   │
│   (based on surveillance)              │
│ • Contains 3-4 strains:                │
│   - 2 Influenza A (H1N1, H3N2)         │
│   - 1-2 Influenza B                    │
│                                        │
│ SPECIAL POPULATIONS:                   │
│ • Pregnancy: IIV (any trimester) ⭐     │
│   - Protects mother AND infant         │
│ • ≥65 years: High-dose or adjuvanted ⭐ │
│ • Egg allergy: Recombinant or IIV      │
│   (safe even with severe allergy)      │
│                                        │
│ EFFICACY:                              │
│ • Variable (40-60% typically)          │
│ • Depends on match to circulating      │
│   strains                              │
│ • Even if imperfect match, reduces     │
│   severity and complications ⭐         │
│                                        │
│ RESPONSE TIME:                         │
│ • Protection in ~2 weeks               │
└────────────────────────────────────────┘

CHEMOPROPHYLAXIS:
┌────────────────────────────────────────┐
│ INDICATIONS:                           │
│ • Post-exposure prophylaxis (household │
│   contact, high-risk)                  │
│ • Outbreak control (institutions)      │
│ • High-risk unvaccinated during season │
│ • Immunocompromised (poor vaccine      │
│   response)                            │
│                                        │
│ REGIMEN:                               │
│ • Oseltamivir 75 mg PO daily           │
│ • Duration: 10 days (post-exposure) or │
│   throughout season                    │
│                                        │
│ NOTE: NOT a substitute for vaccination!│
└────────────────────────────────────────┘

INFECTION CONTROL:
• Respiratory hygiene (cover cough)
• Hand hygiene
• Stay home when sick
• Isolate hospitalized patients (droplet
  precautions)
```

## 2. RESPIRATORY SYNCYTIAL VIRUS (RSV) ⭐⭐⭐⭐⭐

### Virology
```
┌────────────────────────────────────────┐
│ Family: Paramyxoviridae                │
│                                        │
│ • Enveloped, negative-sense ssRNA      │
│ • NON-segmented genome                 │
│ • 2 subtypes: A and B                  │
│ • Forms syncytia (multinucleated giant │
│   cells) ⭐                             │
│                                        │
│ SURFACE PROTEINS:                      │
│ • F protein (fusion): Entry, syncytia  │
│   - Target of antibodies (palivizumab) │
│ • G protein: Attachment                │
└────────────────────────────────────────┘
```

### Epidemiology & Transmission
```
MOST IMPORTANT CAUSE: ⭐⭐⭐⭐⭐
• Leading cause of bronchiolitis in
  infants/young children
• Leading cause of viral pneumonia in
  infants
• #1 cause of infant hospitalization

AGE:
• Peak: 2-6 months
• Nearly all children infected by age 2
• Reinfection common (immunity incomplete)

SEASONALITY:
• Temperate: Winter (Nov-March) ⭐
• Predictable annual outbreaks

TRANSMISSION:
• Respiratory droplets
• Direct contact
• Fomites (survives hours on surfaces)
• Highly contagious
• Healthcare-associated outbreaks common
```

### Clinical Manifestations ⭐⭐⭐⭐⭐

```
INFANTS/YOUNG CHILDREN:
┌────────────────────────────────────────┐
│ BRONCHIOLITIS: ⭐⭐⭐⭐⭐ (Most common) │
│ Age: <2 years (peak 2-6 months)        │
│                                        │
│ PROGRESSION:                           │
│ • URI prodrome (2-4 days)              │
│   - Rhinorrhea, cough, low fever       │
│   ↓                                    │
│ • Lower respiratory symptoms           │
│   - Tachypnea ⭐                        │
│   - Wheezing ⭐⭐⭐⭐                    │
│   - Crackles (fine)                    │
│   - Increased work of breathing        │
│   - Nasal flaring, retractions         │
│   - Grunting                           │
│   - Hypoxemia                          │
│   - Feeding difficulty ⭐               │
│   - Apnea (especially <6 months) ⭐     │
│                                        │
│ PEAK: Day 3-5                          │
│ DURATION: 7-14 days                    │
│                                        │
│ CXR:                                   │
│ • Hyperinflation ⭐                     │
│ • Peribronchial thickening             │
│ • Atelectasis                          │
│ • Patchy infiltrates                   │
│                                        │
│ SEVERE DISEASE RISK FACTORS: ⭐⭐⭐⭐    │
│ • Age <6 months                        │
│ • Prematurity (<35 weeks)              │
│ • Chronic lung disease (BPD)           │
│ • Congenital heart disease (CHD)       │
│ • Immunodeficiency                     │
│ • Neuromuscular disease                │
│                                        │
│ PNEUMONIA: ⭐⭐⭐⭐                      │
│ • Can occur alone or with bronchiolitis│
│ • Tachypnea, hypoxemia                 │
│ • Interstitial or alveolar infiltrates │
└────────────────────────────────────────┘

OLDER CHILDREN/ADULTS:
• Mild URI (common cold)
• Bronchitis, tracheobronchitis
• Pneumonia (elderly, immunocompromised)
• Exacerbation of asthma/COPD
```

### Diagnosis ⭐⭐⭐⭐
```
┌────────────────────────────────────────┐
│ CLINICAL (Bronchiolitis):              │
│ • Often sufficient                     │
│ • Winter season, age <2 years          │
│ • First episode of wheezing            │
│                                        │
│ RAPID ANTIGEN TEST:                    │
│ • Nasopharyngeal swab/aspirate         │
│ • Immunofluorescence (DFA) or EIA      │
│ • Results in hours                     │
│ • Moderate sensitivity/specificity     │
│                                        │
│ RT-PCR: ⭐⭐⭐⭐                         │
│ • Nasopharyngeal swab                  │
│ • Most sensitive/specific              │
│ • Gold standard                        │
│ • Multiplex respiratory panel          │
│                                        │
│ CULTURE:                               │
│ • Slow (days), not for acute management│
│                                        │
│ CXR:                                   │
│ • Not routine for bronchiolitis        │
│ • If severe, concern for pneumonia,    │
│   foreign body, CHF                    │
└────────────────────────────────────────┘
```

### Treatment ⭐⭐⭐⭐⭐

```
BRONCHIOLITIS (AAP Guidelines):
┌────────────────────────────────────────┐
│ SUPPORTIVE CARE ONLY: ⭐⭐⭐⭐⭐         │
│                                        │
│ RECOMMENDED:                           │
│ • Hydration (oral or IV if needed)     │
│ • Supplemental oxygen if hypoxemia     │
│   (SpO2 <90%)                          │
│ • Nasal suctioning (saline drops +     │
│   bulb suction)                        │
│ • Monitor respiratory status, feeding  │
│                                        │
│ NOT RECOMMENDED: ⭐⭐⭐⭐⭐              │
│ • Bronchodilators (albuterol) ⭐        │
│   - No proven benefit                  │
│   - May trial, but discontinue if no   │
│     response                           │
│ • Corticosteroids ⭐                    │
│   - No benefit                         │
│ • Antibiotics ⭐                        │
│   - Unless bacterial superinfection    │
│ • Chest physiotherapy                  │
│ • Ribavirin                            │
│   - Only for severe cases in           │
│     immunocompromised (controversial)  │
│                                        │
│ ADMISSION CRITERIA:                    │
│ • Hypoxemia (SpO2 <90% on room air)    │
│ • Respiratory distress                 │
│ • Dehydration, unable to feed          │
│ • Apnea                                │
│ • Age <3 months (consider)             │
│ • High-risk conditions                 │
└────────────────────────────────────────┘

SEVERE/IMMUNOCOMPROMISED:
• Ribavirin (aerosolized) - controversial
• Only for severe immunocompromised,
  transplant patients
```

### Prevention ⭐⭐⭐⭐⭐

```
PASSIVE IMMUNIZATION:
┌────────────────────────────────────────┐
│ PALIVIZUMAB (Synagis): ⭐⭐⭐⭐⭐        │
│                                        │
│ • Humanized monoclonal antibody        │
│ • Binds RSV F protein                  │
│ • Passive immunity                     │
│                                        │
│ INDICATIONS (AAP): ⭐⭐⭐⭐              │
│ During first year of life:             │
│ • Preterm infants <29 weeks            │
│ • Chronic lung disease of prematurity  │
│   (CLD, requires O2 at 28 days)        │
│ • Hemodynamically significant CHD      │
│                                        │
│ During second year:                    │
│ • CLD requiring medical support        │
│ • Severe immunodeficiency              │
│                                        │
│ DOSING:                                │
│ • 15 mg/kg IM monthly ⭐                │
│ • During RSV season (5 doses max)      │
│ • Nov-March (Northern hemisphere)      │
│                                        │
│ EFFICACY:                              │
│ • 50-55% reduction in hospitalization  │
│ • Does NOT prevent infection           │
│ • Reduces severity                     │
│                                        │
│ EXPENSIVE: ⭐                           │
│ • ~$1000-2000 per dose                 │
│ • Reserved for high-risk only          │
└────────────────────────────────────────┘

NEW PREVENTION (2023):
┌────────────────────────────────────────┐
│ NIRSEVIMAB (Beyfortus):                │
│ • Long-acting monoclonal antibody      │
│ • Single dose protection (entire season)│
│ • Recommended for ALL infants <8 months│
│   entering first RSV season            │
│ • Dose: 50 mg (<5 kg) or 100 mg (≥5 kg)│
│ • Game-changer for prevention          │
│                                        │
│ MATERNAL RSV VACCINE (RSVpreF):        │
│ • Given to pregnant women (32-36 weeks)│
│ • Protects infant first 6 months       │
│ • Alternative to nirsevimab            │
└────────────────────────────────────────┘

INFECTION CONTROL:
• Hand hygiene (most important) ⭐
• Contact precautions (hospitalized)
• Avoid exposure (sick contacts)
• No daycare when sick
```

## 3. PARAINFLUENZA VIRUSES ⭐⭐⭐⭐

### Virology
```
Family: Paramyxoviridae (same as RSV)

• Enveloped, negative-sense ssRNA
• 4 serotypes: Types 1, 2, 3, 4
  - Type 1: Most common, croup
  - Type 2: Croup
  - Type 3: Bronchiolitis, pneumonia (infants)
  - Type 4: Mild URI
```

### Clinical Manifestations ⭐⭐⭐⭐⭐

```
CROUP (Laryngotracheobronchitis): ⭐⭐⭐⭐⭐
┌────────────────────────────────────────┐

## 4. ADENOVIRUS ⭐⭐⭐⭐

### Virology
```
Family: Adenoviridae

• NON-enveloped, dsDNA ⭐
• >50 serotypes

│ • Fever                                │
│ • Children, outbreaks (pools, camps)   │
│                                        │
│ ACUTE RESPIRATORY DISEASE (ARD):       │
│ • Military recruits (crowded)          │
│ • Fever, pharyngitis, cough            │
│ • Pneumonia (can be severe)            │
│                                        │
│ PERTUSSIS-LIKE SYNDROME:               │
│ • Infants                              │
│ • Prolonged cough                      │
└────────────────────────────────────────┘

OCULAR:
┌────────────────────────────────────────┐
│ EPIDEMIC KERATOCONJUNCTIVITIS: ⭐⭐⭐⭐  │
│ • Highly contagious                    │
│ • Watery discharge, photophobia        │
│ • Preauricular lymphadenopathy         │
│ • Can cause corneal infiltrates        │
│ • Healthcare outbreaks (ophthalmology) │
│ • Transmission: Fomites, instruments   │
└────────────────────────────────────────┘

GASTROINTESTINAL:
• Gastroenteritis (serotypes 40, 41)
• Infants/young children

• Hemorrhagic cystitis (children)

### Diagnosis & Treatment
```
DIAGNOSIS:
• PCR (respiratory, stool, conjunctival swab)
• Viral culture
• Antigen detection

TREATMENT:

• Cidofovir (severe immunocompromised)

PREVENTION:
• Hand hygiene
• Disinfection (surfaces, instruments)
• Adenovirus vaccine (military only)
  - Oral, types 4 and 7
  - Not for general public
```

### Q1. Which virus causes pandemics via antigenic shift?
A) Influenza B
B) Influenza A
C) RSV
D) Parainfluenza

**Answer: B**
**Explanation**: Influenza A - segmented genome allows reassortment (antigenic shift) → new H/N combinations → pandemics.

### Q2. Pigs act as "mixing vessel" for influenza because:
A) They have both avian and human receptors
B) They are immunodeficient
C) They have slow viral clearance
D) They shed virus for months

**Answer: A**
**Explanation**: Pigs have receptors for both human and avian influenza → co-infection → reassortment → antigenic shift.

### Q3. Pregnant woman with influenza. Best treatment?
A) Zanamivir (inhaled)
B) Oseltamivir (oral)
C) Supportive care only
D) Amantadine

**Answer: B**
**Explanation**: Oseltamivir (Tamiflu) is safe in pregnancy and preferred neuraminidase inhibitor for influenza treatment.

### Q4. Leading cause of bronchiolitis in infants?
A) Influenza
B) RSV
C) Parainfluenza
D) Adenovirus

**Answer: B**
**Explanation**: RSV is the most common cause of bronchiolitis and viral pneumonia in infants.

### Q5. A 4-month-old with wheezing and hyperinflation on CXR. Treatment?
A) Albuterol
B) Corticosteroids
C) Supportive care only
D) Antibiotics

**Answer: C**
**Explanation**: RSV bronchiolitis - supportive care only per AAP guidelines. Bronchodilators and steroids not beneficial.

### Q6. Palivizumab (Synagis) is given to:

**Answer: B**

## CLINICAL PEARLS ⭐

1. **Influenza = abrupt onset** (vs gradual in common cold)
2. **Severe myalgias = think influenza** (not typical of other URI)
3. **Antigenic shift = pandemics** (only Influenza A, segmented genome)
4. **Start oseltamivir within 48h** for best efficacy (but give later if high-risk/hospitalized)
5. **Oseltamivir safe in pregnancy** - always treat pregnant women with influenza
6. **Annual flu vaccine for everyone** ≥6 months
7. **RSV = winter wheezing in infants** (<2 years)
8. **Bronchiolitis = supportive care only** (no bronchodilators/steroids benefit)
9. **Palivizumab = expensive** (~$1000-2000/dose) - only for high-risk infants
10. **Croup = barking cough + stridor** (parainfluenza most common)
11. **All croup patients get dexamethasone** (0.6 mg/kg single dose)
12. **Steeple sign = croup** (subglottic narrowing)
13. **Adenovirus outbreaks in pools/camps** (survives on surfaces for weeks)
14. **Rhinovirus = most common cold** (>100 serotypes, no vaccine possible)
15. **Common cold usually afebrile** (vs influenza: high fever)
16. **COVID-19: Anosmia/ageusia** distinctive (not typical of other URI)
17. **Secondary bacterial pneumonia after influenza** - S. pneumoniae most common
18. **Improvement then worsening = secondary bacterial pneumonia**
19. **Never give aspirin for viral illness in children** (Reye syndrome)
20. **Hand hygiene most important** for preventing respiratory virus transmission
