# Herpesviruses

## Overview
Herpesviridae is a family of large, enveloped, double-stranded DNA viruses characterized by **latency** (lifelong persistence) and ability to reactivate. Eight human herpesviruses (HHV) have been identified.

---

## HERPESVIRUS FAMILY OVERVIEW ⭐⭐⭐⭐⭐

### General Characteristics
```
Common Features:
┌────────────────────────────────────────┐
│ STRUCTURE:                             │
│ • Large enveloped DNA virus            │
│ • Double-stranded linear DNA           │
│ • Icosahedral capsid                   │
│                                        │
│ BIOLOGY:                               │
│ • Replicate in nucleus                 │
│ • Produce intranuclear inclusions      │
│ • Establish LATENCY ⭐⭐⭐⭐⭐          │
│ • Lifelong persistence                 │
│ • Reactivation with stress,            │
│   immunosuppression                    │
│                                        │
│ TRANSMISSION:                          │
│ • Direct contact (saliva, genital,     │
│   respiratory secretions, blood)       │
│ • Vertical (transplacental)            │
└────────────────────────────────────────┘
```

### Classification (3 Subfamilies)
```
┌────────────────────────────────────────┐
│ α-HERPESVIRUSES (Neurotropic):         │
│ • Rapid replication, lytic             │
│ • Latency in neurons                   │
│                                        │
│ HHV-1: Herpes Simplex Virus-1 (HSV-1)  │
│ HHV-2: Herpes Simplex Virus-2 (HSV-2)  │
│ HHV-3: Varicella-Zoster Virus (VZV)    │
├────────────────────────────────────────┤
│ β-HERPESVIRUSES (Lymphotropic):        │
│ • Slow replication                     │
│ • Latency in lymphocytes, monocytes    │
│ • Immunosuppressive                    │
│                                        │
│ HHV-5: Cytomegalovirus (CMV)           │
│ HHV-6: Roseolovirus (roseola)          │
│ HHV-7: Similar to HHV-6                │
├────────────────────────────────────────┤
│ γ-HERPESVIRUSES (Lymphotropic):        │
│ • Latency in lymphocytes               │
│ • Oncogenic potential                  │
│                                        │
│ HHV-4: Epstein-Barr Virus (EBV)        │
│ HHV-8: Kaposi Sarcoma Herpesvirus      │
│        (KSHV)                          │
└────────────────────────────────────────┘
```

## 1. HERPES SIMPLEX VIRUSES (HSV-1 & HSV-2) ⭐⭐⭐⭐⭐

### Virology
- **HSV-1**: Primarily orofacial infections
- **HSV-2**: Primarily genital infections
- **However**: Either can infect either site (overlap increasing)
- **Latency**: Dorsal root ganglia or trigeminal ganglia

### Transmission
- **HSV-1**: Oral secretions, childhood acquisition common
- **HSV-2**: Sexual contact, vertical transmission (birth canal)
- **Asymptomatic shedding**: Can transmit without symptoms ⭐

### Clinical Manifestations

#### A. Primary Infections
```
OROFACIAL (HSV-1 > HSV-2):
┌────────────────────────────────────────┐
│ GINGIVOSTOMATITIS (children): ⭐        │
│ • Most common primary HSV-1            │
│ • Painful vesicles → ulcers            │
│ • Gums, tongue, lips, oral mucosa      │
│ • Fever, malaise, lymphadenopathy      │
│ • Self-limited (7-14 days)             │
│ • Often subclinical in adults          │
│                                        │
│ PHARYNGITIS/TONSILLITIS:               │
│ • Mimics streptococcal pharyngitis     │
│ • Exudative, vesicular lesions         │
└────────────────────────────────────────┘

GENITAL HERPES (HSV-2 > HSV-1): ⭐⭐⭐⭐⭐
┌────────────────────────────────────────┐
│ PRIMARY (severe):                      │
│ • Incubation: 2-12 days                │
│ • Painful grouped vesicles on          │
│   erythematous base                    │
│ • Progress to pustules → ulcers        │
│   → crusting                           │
│ • Bilateral, extensive                 │
│ • Dysuria, urinary retention           │
│ • Tender inguinal lymphadenopathy      │
│ • Systemic symptoms (fever, headache)  │
│ • Duration: 2-3 weeks                  │
│                                        │
│ Males: Glans, shaft, prepuce           │
│ Females: Vulva, vagina, cervix (90%)   │
│                                        │
│ Complications:                         │
│ • Aseptic meningitis (10-30%)          │
│ • Autonomic dysfunction (urinary       │
│   retention, constipation)             │
│ • Sacral radiculopathy                 │
└────────────────────────────────────────┘
```

#### B. Recurrent Infections
```

┌────────────────────────────────────────┐
│ • HSV-1 reactivation from trigeminal   │
│   ganglion                             │
│ • Prodrome: Tingling, burning (24h)    │
│ • Vesicles at mucocutaneous junction   │
│   (vermilion border of lips)           │
│ • Smaller, unilateral                  │
│ • Self-limited (7-10 days)             │
│                                        │
│ Triggers:                              │
│ • UV exposure (sunlight) ⭐             │
│ • Fever, stress, menstruation          │
│ • Immunosuppression                    │
│ • Trauma                               │
└────────────────────────────────────────┘

RECURRENT GENITAL HERPES:
┌────────────────────────────────────────┐
│ • Less severe than primary             │
│ • Prodrome (tingling)                  │
│ • Fewer lesions, unilateral            │
│ • Shorter duration (5-10 days)         │
│ • No systemic symptoms                 │
│ • HSV-2: 90% recurrence rate           │
│ • HSV-1 genital: Lower recurrence      │
│                                        │
│ Frequency: Variable (HSV-2 > HSV-1)    │
│ • Average: 4 episodes/year (HSV-2)     │
│ • Decreases over time                  │
└────────────────────────────────────────┘
```

```

┌────────────────────────────────────────┐

│ • Painful vesicles, erythema, swelling │
│ • Risk: Healthcare workers (before     │

└────────────────────────────────────────┘

ECZEMA HERPETICUM (Kaposi Varicelliform Eruption):
┌────────────────────────────────────────┐
│ • HSV superinfection of eczematous skin│
│ • Widespread vesiculopustular eruption │
│ • "Punched-out" lesions                │
│ • Fever, malaise                       │
│ • Can be life-threatening              │
│ • Requires systemic antivirals         │
└────────────────────────────────────────┘

HSV ENCEPHALITIS: ⭐⭐⭐⭐⭐
┌────────────────────────────────────────┐
│ • Most common sporadic viral           │
│   encephalitis (adults)                │
│ • HSV-1 (>90%), any age                │
│ • TEMPORAL LOBE predilection ⭐         │
│                                        │
│ Clinical:                              │
│ • Fever, headache                      │
│ • Altered mental status, seizures      │
│ • Focal neurologic signs               │

│ • Olfactory/gustatory hallucinations   │
│                                        │

│ • MRI: Temporal lobe abnormalities ⭐   │
│   (hemorrhagic necrosis)               │

│                                        │
│ Treatment: ⭐⭐⭐⭐⭐                     │
│ • IV Acyclovir 10 mg/kg Q8h × 14-21d   │
│ • START EMPIRICALLY (don't wait for    │
│   confirmation)                        │
│ • Mortality 70% untreated, 30% treated │
│ • Neurologic sequelae common           │
└────────────────────────────────────────┘

NEONATAL HERPES: ⭐⭐⭐⭐⭐
┌────────────────────────────────────────┐
│ Transmission:                          │
│ • Intrapartum (85%) - birth canal ⭐    │
│ • Postnatal (10%) - kissing, caregivers│
│ • Intrauterine (5%) - transplacental   │
│                                        │
│ Highest risk:                          │
│ • Primary maternal infection near      │
│   delivery (30-50% transmission)       │
│ • Recurrent: <3% risk                  │
│                                        │
│ Three Forms:                           │
│ 1. SEM (Skin-Eye-Mouth): 45%           │
│    • Best prognosis if treated         │
│    • Vesicular rash, keratitis         │
│                                        │
│ 2. CNS Disease: 30%                    │
│    • Encephalitis, seizures            │
│    • High morbidity/mortality          │
│                                        │
│ 3. Disseminated: 25%                   │
│    • Multi-organ (liver, lungs, CNS)   │
│    • Highest mortality (30%)           │
│    • Sepsis-like, DIC                  │
│                                        │
│ Diagnosis:                             │
│ • HSV PCR from lesions, CSF, blood     │
│ • Surface swabs (mouth, nasopharynx,   │
│   conjunctiva, rectum) at 24h of life  │
│                                        │
│ Treatment:                             │
│ • IV Acyclovir 20 mg/kg Q8h            │
│ • Duration: 14d (SEM), 21d (CNS/Diss)  │
│ • Oral suppression × 6 months after    │
│                                        │
│ Prevention:                            │
│ • C-section if active lesions at labor │
│ • Suppressive acyclovir from 36 weeks  │
│   if history of genital HSV            │
└────────────────────────────────────────┘
```

### Diagnosis
```
┌────────────────────────────────────────┐
│ VIRAL CULTURE:                         │
│ • Gold standard (historically)         │
│ • Vesicle fluid best specimen          │
│ • CPE: Multinucleated giant cells      │
│ • Sensitivity decreases with time      │
│   (best if <48h)                       │
│                                        │
│ PCR: ⭐⭐⭐⭐⭐                           │
│ • Most sensitive and specific          │
│ • Preferred for CNS infections (CSF)   │
│ • Quantitative possible                │
│                                        │
│ DIRECT FLUORESCENT ANTIBODY (DFA):     │
│ • Rapid (hours)                        │
│ • Less sensitive than PCR              │
│                                        │
│ TZANCK SMEAR (Historical):             │
│ • Scrape base of vesicle               │
│ • Giemsa stain                         │
│ • Multinucleated giant cells ⭐         │
│ • Cannot differentiate HSV from VZV    │
│ • Low sensitivity (60%)                │
│                                        │
│ SEROLOGY:                              │
│ • Type-specific IgG (HSV-1 vs HSV-2)   │
│ • Not for acute diagnosis              │
│ • Useful for asymptomatic screening    │
└────────────────────────────────────────┘
```

### Treatment ⭐⭐⭐⭐⭐
```
ORAL ANTIVIRALS (Immunocompetent):
┌────────────────────────────────────────┐
│ PRIMARY GENITAL HERPES:                │
│ • Acyclovir 400 mg PO TID × 7-10 days  │
│ • Valacyclovir 1 g PO BID × 7-10 days  │
│ • Famciclovir 250 mg PO TID × 7-10 days│
│                                        │
│ RECURRENT GENITAL HERPES:              │
│ Episodic (at first sign):              │
│ • Acyclovir 800 mg PO TID × 2 days     │
│ • Valacyclovir 500 mg PO BID × 3 days  │
│ • Famciclovir 1000 mg PO BID × 1 day   │
│                                        │
│ Suppressive (frequent recurrences):    │
│ • Acyclovir 400 mg PO BID daily        │
│ • Valacyclovir 500 mg PO daily         │
│ • Reduces recurrences by 70-80%        │
│ • Reduces transmission risk by 50%     │
│                                        │
│ OROLABIAL HERPES (Recurrent):          │
│ • Valacyclovir 2 g PO Q12h × 1 day     │
│ • Famciclovir 1500 mg single dose      │
│ • Topical acyclovir/penciclovir        │
│   (minimal benefit)                    │
└────────────────────────────────────────┘

IV ANTIVIRALS (Severe):
┌────────────────────────────────────────┐
│ HSV ENCEPHALITIS:                      │
│ • Acyclovir 10 mg/kg IV Q8h × 14-21d ⭐│
│                                        │
│ NEONATAL HERPES:                       │
│ • Acyclovir 20 mg/kg IV Q8h            │
│   - 14 days (SEM)                      │
│   - 21 days (CNS or disseminated)      │
│                                        │
│ IMMUNOCOMPROMISED:                     │
│ • Acyclovir 5-10 mg/kg IV Q8h          │
│                                        │
│ ACYCLOVIR-RESISTANT (rare):            │
│ • Foscarnet 40 mg/kg IV Q8h            │
│   (immunocompromised, prolonged        │
│    acyclovir use)                      │
└────────────────────────────────────────┘
```

### Prevention
- **Condoms**: Reduce but don't eliminate transmission (asymptomatic shedding)
- **Suppressive therapy**: Reduces transmission risk by ~50%
- **Avoid sex during outbreaks**
- **C-section**: If active genital lesions at delivery
- **Suppressive acyclovir in pregnancy**: 36 weeks onward if history of genital HSV

## 2. VARICELLA-ZOSTER VIRUS (VZV / HHV-3) ⭐⭐⭐⭐⭐

### Virology
- **Primary infection**: Varicella (chickenpox)
- **Reactivation**: Herpes zoster (shingles)
- **Latency**: Dorsal root ganglia and cranial nerve ganglia
- **Highly contagious**: Airborne and contact transmission

### Varicella (Chickenpox) ⭐⭐⭐⭐⭐

#### Clinical Features
```
Incubation: 10-21 days (average 14-16)

RASH EVOLUTION (Pathognomonic): ⭐⭐⭐⭐⭐
"Dew drop on rose petal"
┌────────────────────────────────────────┐
│ Macule → Papule → Vesicle → Pustule   │
│ → Crust (4-5 stages)                   │
│                                        │
│ Characteristic: "Crops" ⭐              │
│ • New lesions appear in waves          │
│ • Different stages present together    │
│   (unlike smallpox - all same stage)   │
│ • Start on face/trunk → centripetal    │
│   spread                               │
│                                        │
│ Distribution:                          │
│ • Centripetal (trunk > extremities)    │
│ • Also scalp, mucous membranes         │
│ • Typically 200-500 lesions            │
│ • Pruritic (very itchy!) ⭐             │
│                                        │
│ Systemic:                              │
│ • Fever (usually 1-2 days before rash) │
│ • Malaise, headache                    │
│ • Duration: 5-7 days                   │
└────────────────────────────────────────┘

CONTAGIOUS PERIOD:
• 1-2 days before rash until all lesions crusted
• Typically 5-7 days after rash onset
```

#### Complications
```
CHILDREN (Usually mild):
┌────────────────────────────────────────┐
│ • Secondary bacterial infection ⭐      │
│   (S. aureus, Group A Strep)           │
│   - Impetigo, cellulitis, abscess      │
│                                        │
│ • Cerebellar ataxia (most common       │
│   CNS complication in children)        │
│   - Resolves spontaneously             │
│                                        │
│ • Reye syndrome (aspirin use) ⭐        │
│   - Encephalopathy + fatty liver       │
│   - NEVER give aspirin in chickenpox!  │
└────────────────────────────────────────┘

ADULTS (More severe):
┌────────────────────────────────────────┐
│ • Varicella pneumonia ⭐⭐⭐⭐            │
│   - 10-20% in adults                   │
│   - Respiratory failure, ARDS          │
│   - CXR: Diffuse nodular infiltrates   │
│   - High mortality                     │
│                                        │
│ • Encephalitis                         │
│ • Hepatitis                            │
│ • Thrombocytopenia                     │
└────────────────────────────────────────┘

IMMUNOCOMPROMISED:
┌────────────────────────────────────────┐
│ • Disseminated/severe disease          │
│ • Prolonged new lesion formation       │
│ • Visceral involvement (liver, lungs)  │
│ • Hemorrhagic complications            │
│ • High mortality without treatment     │
└────────────────────────────────────────┘

PREGNANCY: ⭐⭐⭐⭐⭐
┌────────────────────────────────────────┐
│ MATERNAL:                              │
│ • Varicella pneumonia (10-20%)         │
│ • Higher mortality than non-pregnant   │
│                                        │
│ FETAL (Congenital Varicella Syndrome): │
│ • If infection <20 weeks: 0.4-2% risk  │
│ • Limb hypoplasia, scarring            │
│ • Neurologic abnormalities             │
│ • Eye defects (chorioretinitis)        │
│ • Low birth weight                     │
│                                        │
│ NEONATAL VARICELLA:                    │
│ • If maternal rash 5 days before to    │
│   2 days after delivery → 17-30%       │
│   mortality (no maternal antibodies)   │
│ • Disseminated, severe                 │
│ • Give VZIG to neonate if maternal     │
│   rash in this window                  │
└────────────────────────────────────────┘
```

### Herpes Zoster (Shingles) ⭐⭐⭐⭐⭐

#### Clinical Features
```
Pathogenesis:
• Reactivation of latent VZV from dorsal
  root ganglia
• Lifetime risk: 30% (increases with age)
• Most common in >50 years, immunocompromised

CLASSIC PRESENTATION: ⭐⭐⭐⭐⭐
┌────────────────────────────────────────┐
│ PRODROME (2-3 days before rash):       │
│ • Dermatomal pain, burning, tingling   │
│ • Hyperesthesia                        │
│                                        │
│ RASH:                                  │
│ • Unilateral, dermatomal ⭐⭐⭐⭐⭐      │
│   (does NOT cross midline)             │
│ • Most common: Thoracic (50%)          │
│ • Grouped vesicles on erythematous base│
│ • "Zosteriform" distribution           │
│ • Crusts in 7-10 days                  │
│                                        │
│ PAIN:                                  │
│ • Severe, burning, shooting            │
│ • Can be debilitating                  │
└────────────────────────────────────────┘

SPECIAL FORMS:
┌────────────────────────────────────────┐
│ HERPES ZOSTER OPHTHALMICUS: ⭐⭐⭐⭐     │
│ • CN V1 (ophthalmic division)          │
│ • Hutchinson's sign: Lesions on tip/   │
│   side of nose → ocular involvement ⭐  │
│ • Complications: Keratitis, uveitis,   │
│   vision loss                          │
│ • URGENT ophthalmology referral        │
│                                        │
│ RAMSAY HUNT SYNDROME: ⭐⭐⭐⭐           │
│ • Geniculate ganglion (CN VII)         │
│ • Triad:                               │
│   1. Facial nerve palsy (LMN type)     │
│   2. Ear pain                          │
│   3. Vesicles in ear canal/tympanic    │
│      membrane                          │
│ • Vertigo, tinnitus, hearing loss      │
│ • Worse prognosis than Bell's palsy    │
│                                        │
│ DISSEMINATED ZOSTER:                   │
│ • >20 lesions outside primary/adjacent │
│   dermatomes                           │
│ • Immunocompromised patients           │
│ • Visceral involvement possible        │
└────────────────────────────────────────┘
```

#### Complications
```
POST-HERPETIC NEURALGIA (PHN): ⭐⭐⭐⭐⭐
┌────────────────────────────────────────┐
│ • Most common complication             │
│ • Persistent pain >90 days after rash  │
│ • Incidence increases with age:        │
│   - <40 years: <10%                    │
│   - >60 years: 40-50%                  │
│   - >70 years: >50%                    │
│                                        │
│ Characteristics:                       │
│ • Burning, stabbing, aching pain       │
│ • Allodynia (pain from non-painful     │
│   stimuli)                             │
│ • Can last months to years             │
│ • Significantly impacts quality of life│
│                                        │
│ Prevention:                            │
│ • Early antiviral treatment ⭐          │
│   (within 72h reduces risk)            │
│ • Vaccination (Shingrix)               │
│                                        │
│ Treatment:                             │
│ • Gabapentin, pregabalin               │
│ • Tricyclic antidepressants            │
│   (amitriptyline, nortriptyline)       │
│ • Topical lidocaine patches            │
│ • Capsaicin cream                      │
│ • Opioids (severe cases)               │
└────────────────────────────────────────┘

Other Complications:
• Bacterial superinfection
• Motor neuropathy (rare)
• Meningoencephalitis
• Myelitis
```

### Diagnosis
```
CLINICAL (usually sufficient):
• Chickenpox: Characteristic rash with crops
• Zoster: Unilateral dermatomal vesicular rash

LABORATORY (if needed):
┌────────────────────────────────────────┐
│ • PCR (vesicle fluid, CSF): ⭐          │
│   Most sensitive/specific              │
│                                        │
│ • DFA (Direct Fluorescent Antibody):   │
│   Rapid, scraping from lesion base     │
│                                        │
│ • Tzanck smear: Multinucleated giant   │
│   cells (cannot differentiate HSV/VZV) │
│                                        │
│ • Viral culture: Slow, less sensitive  │
│                                        │
│ • Serology: Not useful acutely         │
│   (IgG: immunity/prior infection)      │
└────────────────────────────────────────┘
```

### Prevention ⭐⭐⭐⭐⭐
```
VARICELLA VACCINE (Live Attenuated):
┌────────────────────────────────────────┐
```

- **T**emporal lobe predilection
- **A**cyclovir IV treatment
- **PCR** of CSF (diagnostic)

### 4. **Chickenpox vs Smallpox - "Chickens Crop, Cows Same"**
- **Chick**enpox: Lesions in different stages (**Crop**s)
- Small**pox**: Lesions all **Same** stage

### 5. **VZV Treatment Timing - "72 hours or else PHN"**
- Start antivirals within **72 hours** of zoster rash
- Reduces **Post-Herpetic Neuralgia** risk

### 6. **EBV Mono Triad - "FPL"**
- **F**ever
- **P**haryngitis
- **L**ymphadenopathy (posterior cervical)

### 7. **EBV + Ampicillin = "Mono Rash"**
- **Mono**nucleosis + **A**mpicillin/amoxicillin
- **R**ash in 90%

### 8. **CMV "Owl's Eye"**
- **O**rgans: Retina, Colon, Esophagus, Lungs, Brain
- **W**BC: Leukopenia
- **L**arge intranuclear inclusions

### 9. **Roseola - "Fever Falls, Rash Rises"**
- High **F**ever 3-5 days
- Fever **F**alls (defervescence)
- **R**ash **R**ises (appears as fever breaks)

### 10. **Kaposi Sarcoma - "Purple People AIDS"**
- **P**urple lesions
- **P**alate (hard palate, oral)
- **AIDS** (HHV-8)

## NEET PG MCQS

### Q1. A 4-year-old has high fever for 3 days, appears well. Fever resolves, then develops maculopapular rash on trunk. Diagnosis?
A) Measles
B) Rubella
C) Roseola infantum
D) Scarlet fever

**Answer: C**
**Explanation**: Roseola (HHV-6/7) - classic "fever then rash" pattern. Child appears well despite high fever, rash appears as fever breaks.

---

### Q2. A patient develops dendritic corneal ulcer. Treatment should include all EXCEPT:
A) Topical trifluridine
B) Topical corticosteroids
C) Systemic acyclovir
D) Fluorescein staining for diagnosis

**Answer: B**
**Explanation**: HSV keratitis - NEVER give steroids (worsen infection). Treat with topical antivirals ± systemic acyclovir.

---

### Q3. Best diagnostic test for HSV encephalitis?
A) Blood culture
B) CSF PCR for HSV DNA
C) Brain biopsy
D) Serum HSV IgM

**Answer: B**
**Explanation**: CSF PCR for HSV DNA is most sensitive/specific for HSV encephalitis. Start empiric acyclovir while awaiting results.

---

### Q4. A neonate born to mother with active genital herpes lesions at delivery. Best management?
A) Observation only
B) Oral acyclovir prophylaxis
C) IV acyclovir 20 mg/kg Q8h
D) VZIG administration

**Answer: C**
**Explanation**: Neonatal herpes - high-risk exposure (active maternal lesions). Start IV acyclovir 20 mg/kg Q8h empirically, test infant (surface cultures, PCR).

---

### Q5. A patient with chickenpox should avoid which medication for fever?
A) Acetaminophen
B) Ibuprofen
C) Aspirin
D) All of the above

**Answer: C**
**Explanation**: NEVER give aspirin in chickenpox (or any viral illness in children) - risk of Reye syndrome (encephalopathy + fatty liver).

---

### Q6. Most common complication of herpes zoster in elderly?
A) Bacterial superinfection
B) Post-herpetic neuralgia
C) Disseminated disease
D) Motor neuropathy

**Answer: B**
**Explanation**: Post-herpetic neuralgia (PHN) - most common complication, especially >60 years (40-50%). Persistent pain >90 days after rash.

---

### Q7. Hutchinson's sign in herpes zoster indicates:
A) Disseminated disease
B) Ophthalmic involvement
C) Ramsay Hunt syndrome
D) Immunosuppression

**Answer: B**
**Explanation**: Hutchinson's sign - vesicles on tip/side of nose in zoster ophthalmicus (CN V1) → high risk of ocular involvement. Needs urgent ophthalmology referral.

---

### Q8. A patient with infectious mononucleosis develops rash after taking amoxicillin. This indicates:
A) Penicillin allergy
B) EBV-related phenomenon, not true allergy
C) Severe drug reaction, avoid all β-lactams
D) Scarlet fever coinfection

**Answer: B**
**Explanation**: EBV mono + ampicillin/amoxicillin → rash in 90%. NOT a true penicillin allergy (mechanism unclear). Can use penicillins in future.

---

### Q9. Heterophile antibody test is positive in:
A) EBV mononucleosis
B) CMV mononucleosis
C) Both
D) Neither

**Answer: A**
**Explanation**: Monospot (heterophile antibody) positive in EBV mono, NEGATIVE in CMV mono (key differentiator).

---

### Q10. "Owl's eye" inclusions are characteristic of:
A) HSV
B) VZV
C) CMV
D) EBV

**Answer: C**
**Explanation**: CMV - large intranuclear inclusions with clear halo ("owl's eye") + cytoplasmic inclusions in enlarged cells (cytomegalic).

---

### Q11. An HIV patient with CD4 40 has "pizza pie" retinal lesions. Treatment?
A) Acyclovir
B) Ganciclovir
C) Foscarnet
D) B or C

**Answer: D**
**Explanation**: CMV retinitis (CD4 <50, characteristic fundoscopic findings). First-line: ganciclovir or valganciclovir. Alternative: foscarnet.

---

### Q12. Congenital CMV diagnosis requires specimen collection within:
A) First 24 hours
B) First 3 weeks
C) First 3 months
D) First year

**Answer: B**
**Explanation**: CMV culture/PCR from urine/saliva within first 3 weeks of life distinguishes congenital from perinatal/postnatal infection.

## REVISION CHECKLIST

### Must Know for NEET PG:
- [ ] All herpesviruses establish **latency** (lifelong persistence, reactivation)
- [ ] α-Herpesviruses (HSV-1, HSV-2, VZV): Neurotropic, latency in ganglia
- [ ] β-Herpesviruses (CMV, HHV-6, HHV-7): Lymphotropic, slow replication
- [ ] γ-Herpesviruses (EBV, HHV-8): Lymphotropic, oncogenic potential
- [ ] HSV-1: Orolabial (primarily), **encephalitis** (temporal lobe, CSF PCR, IV acyclovir 10 mg/kg)
- [ ] HSV-2: Genital (primarily), **neonatal herpes** (intrapartum transmission, IV acyclovir 20 mg/kg)
- [ ] HSV keratitis: Dendritic ulcer, NEVER steroids
- [ ] Neonatal herpes: SEM (skin-eye-mouth), CNS, disseminated; IV acyclovir 20 mg/kg Q8h
- [ ] Tzanck smear: Multinucleated giant cells (HSV/VZV, cannot differentiate)
- [ ] VZV: Chickenpox (primary), Shingles (reactivation)
- [ ] Chickenpox: Crops (different stages together), centripetal, pruritic, NEVER aspirin (Reye)
- [ ] Varicella complications: Pneumonia (adults), bacterial superinfection, cerebellar ataxia (children)
- [ ] Herpes zoster: Unilateral dermatomal, does NOT cross midline
- [ ] Hutchinson's sign: Nose lesions → ophthalmic involvement (urgent referral)
- [ ] Ramsay Hunt: CN VII palsy + ear pain + ear vesicles (VZV)
- [ ] Post-herpetic neuralgia: Most common zoster complication (>60 years: 40-50%), treat zoster within 72h
- [ ] Varicella vaccine: Live attenuated, 12-15 months and 4-6 years
- [ ] Shingrix (recombinant zoster vaccine): ≥50 years, 2 doses, >90% efficacy
- [ ] EBV mono triad: Fever + Pharyngitis + Lymphadenopathy (posterior cervical)
- [ ] EBV + ampicillin/amoxicillin = rash (90%, NOT true allergy)
- [ ] Splenomegaly (50%) → avoid contact sports × 3-4 weeks (rupture risk)
- [ ] Atypical lymphocytosis ≥10% (reactive CD8+ T cells)
- [ ] Monospot (heterophile antibody): Positive in EBV mono, NEGATIVE in CMV mono
- [ ] EBV serology: Acute = VCA-IgM+ VCA-IgG+ EBNA-; Past = VCA-IgG+ EBNA+
- [ ] EBV malignancies: Burkitt lymphoma (endemic), nasopharyngeal carcinoma, Hodgkin lymphoma, PTLD
- [ ] CMV mono: Heterophile-negative, more hepatitis, less pharyngitis/lymphadenopathy
- [ ] CMV "owl's eye" inclusions: Large intranuclear + cytoplasmic, in enlarged cells
- [ ] CMV retinitis: AIDS (CD4 <50), "pizza pie" appearance, painless vision loss
- [ ] CMV colitis: AIDS, bloody diarrhea, colonoscopy ulcers, "owl's eye" on biopsy
- [ ] Congenital CMV: Most common congenital viral infection, periventricular calcifications, sensorineural hearing loss (most common sequela), diagnose within 3 weeks
- [ ] CMV transplant: D+/R- highest risk, prophylaxis with valganciclovir
- [ ] Ganciclovir: CMV treatment, myelosuppression (neutropenia)
- [ ] Foscarnet: CMV alternative, nephrotoxic, electrolyte abnormalities
- [ ] Roseola (HHV-6/7): Fever 3-5 days → rash as fever breaks, febrile seizures common
- [ ] HHV-8: Kaposi sarcoma (AIDS, purple lesions, oral palate), 100% association
